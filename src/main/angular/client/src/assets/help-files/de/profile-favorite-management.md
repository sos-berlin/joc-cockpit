# Profil - Verwaltung Favoriten

Die Registerkarte *Profil - Verwaltung Favoriten* ermöglicht die Angabe von bevorzugten Einträgen auf verschiedenen Seiten

- für die Auswahl von *Agenten*, die Jobs zugewiesen werden.
- für *Eingabeprüfungen*, die Benutzereingaben für Auftragsvariablen regeln.

Einzelheiten finden Sie unter [JS7 - Inventory Favorites](https://kb.sos-berlin.com/display/JS7/JS7+-+Inventory+Favorites) und [JS7 - Profiles - Favorite Management](https://kb.sos-berlin.com/display/JS7/JS7+-+Profiles+-+Favorite+Management).

## Bevorzugte Agenten

Unter [Standalone Agent](/initial-operation-register-agent-standalone) und [Subagent Cluster](/initial-operation-register-agent-subagent-cluster) finden Benutzer möglicherweise eine größere Anzahl von Agenten in ihrer Umgebung. Anstatt durch lange Listen zu blättern, können Benutzer ihre *Bevorzugten Agenten* angeben, die dann ganz oben in der Liste der zuweisbaren Agenten erscheinen.

### Hinzufügen von bevorzugten Agenten

*Bevorzugte Agenten* können direkt bei der Zuweisung von Agenten auf der Registerkarte [Konfiguration - Inventar - Arbeitsablauf - Job-Eigenschaften](/configuration-inventory-workflow-job-properties) hinzugefügt werden. Wenn Sie auf den Link \* für einen Agenten klicken, wird der entsprechende Agent zur Liste der Favoriten hinzugefügt.

Benutzer können *Bevorugte Agenten* von ihrem *Profil* aus über die Schaltfläche *Favoriten hinzufügen* verwalten. Dies ermöglicht die Auswahl des Agenten, der hinzugefügt werden soll.

### Liste der bevorzugten Agenten

Die Liste der *Bevorzugten Agenten* kann über die Schaltfläche *Mit mir geteilt* eingeschränkt werden. Dadurch wird die Liste auf die von anderen Benutzern freigegebenen Favoriten beschränkt.

Sie können die Liste weiter einschränken, indem Sie eine Zeichenfolge für das Eingabefeld *Suche* angeben, die im Namen des Agenten gesucht wird. Die Suche impliziert eine linke und rechte Trunkierung.

Benutzer können die Liste ordnen, indem sie einen *Bevorzugten Agenten* an eine frühere oder spätere Position verschieben.

### Operationen für bevorzugte Agenten

Die folgenden Operationen sind über das 3-Punkte Aktionsmenü eines *Beovrzugten Agenten* möglich:

- **Bearbeiten** ermöglicht es, den Agenten zu wechseln.
- **Freigeben** bietet die Möglichkeit, den Favoriten mit anderen Benutzern zu teilen. Geteilte Favoriten sind durch ein entsprechendes Symbol gekennzeichnet.
- **Löschen** entfernt den entsprechenden Agenten aus der Liste der Favoriten.

## Bevorzugte Eingabeprüfungen

Benutzer können eine größere Anzahl von *Eingabeprüfungen* in ihrem Inventar haben, die auf [Konfiguration - Inventar - Arbeitsabläufe](/configuration-inventory-workflows) angewendet werden. Wenn ein Arbeitsablauf Variablen vom Datentyp Zeichenkette angibt, kann eine *Eingabeprüfung* angewendet werden, um die Benutzereingabe einzuschränken. *Eingabeprüfungen* sind reguläre Ausdrücke, die angewendet werden, um passende Benutzereingaben zu prüfen.

Bei der Zuweisung von *Eingabeprüfungen* zu Arbeitsablauf-Variablen können die Benutzer ihre *Bevorzugten Eingabeprüfungen* angeben, die ganz oben auf der Liste erscheinen, anstatt durch lange Listen zu blättern.

### Hinzufügen von bevorzugten Eingabeprüfungen

*Bevorzugte Eingabeprüfungen* können direkt bei der Angabe von Arbeitsablauf-Variablen in der Ansicht [Konfiguration - Inventar - Arbeitsabläufe](/configuration-inventory-workflows) hinzugefügt werden. Wenn Sie auf den Link \* für eine *Eingabeprüfung* klicken, wird die zugehörige *Eingabeprüfung* zur Liste der Favoriten hinzugefügt.

Benutzer können *Bevorzugte Eingabeprüfungen* aus ihrem *Profil* über die Schaltfläche *Favoriten hinzufügen* verwalten. Dabei können Sie den Namen der *Eingabeprüfung* und den zugehörigen regulären Ausdruck angeben, der hinzugefügt werden soll.

### Liste der bevorzugten Eingabeprüfungen

Die Liste der *Bevorzugten Eingabeprüfungen* kann über die Schaltfläche *Mit mir geteilt* eingeschränkt werden. Dadurch wird die Liste auf die von anderen Benutzern freigegebenen Favoriten beschränkt.

Benutzer können die Liste weiter einschränken, indem sie eine Zeichenfolge für das Eingabefeld *Suche* angeben, die im Namen der *Eingabeprüfung* gesucht wird. Die Suche impliziert eine linke und rechte Trunkierung.

Benutzer können die Liste ordnen, indem sie eine *Bevorzugte Eingabeprüfung* an eine frühere oder spätere Position verschieben.

### Operationen für bevorzugte Eingabeprüfungen

Die folgenden Operationen sind über das 3-Punkte Aktionsmenü einer *Bevorzugten Eingabeprüfung* verfügbar:

- **Bearbeiten** ermöglicht die Änderung des Namens und des regulären Ausdrucks einer Eingabeprüfung.
- **Freigeben** bietet die Möglichkeit, den Favoriten mit anderen Benutzern zu teilen. Geteilte Favoriten sind durch ein entsprechendes Symbol gekennzeichnet.
- **Löschen** entfernt die entsprechende *Eingabeprüfung* aus der Liste der Favoriten.
