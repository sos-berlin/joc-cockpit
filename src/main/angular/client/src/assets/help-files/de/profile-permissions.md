# Profil - Berechtigungen

Die Registerkarte *Profil - Berechtigungen* zeigt die Berechtigungen der Rollen an, die einem Benutzerkonto zugewiesen sind.

## Rollen

Es werden die Rollen angezeigt, die dem Benutzerkonto zugewiesen wurden. Die daraus resultierenden Berechtigungen werden aus den Rollenzuweisungen zusammengeführt.

## Berechtigungen

### JOC Cockpit Berechtigungen

Die Berechtigungen werden für alle JOC Cockpit Instanzen angezeigt.
 
### Controller Berechtigungen

Die Berechtigungen werden pro Controller angezeigt. Die *Standard* Controller Berechtigungen gelten, wenn keine spezifischen Berechtigungen für den entsprechenden Controller konfiguriert sind.
