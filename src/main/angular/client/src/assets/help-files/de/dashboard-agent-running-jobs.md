# Agenten, die Aufträge ausführen

Das Panel *Agent Running Jobs* zeigt die Anzahl der laufenden Aufträge pro Agent an.

<img src="dashboard-agent-running-jobs.png" alt="Agent Running Jobs" width="350" height="140" />

## Laufende Aufträge

Die Informationen sind dynamisch und können sich alle 2-3 Sekunden ändern. Die Sortierung der Agenten wird von der höchsten zur niedrigsten Anzahl von Jobs angepasst.

Das Panel zählt nur laufende Aufträge, Aufträge, die andere Workflow-Anweisungen passieren, werden nicht gezählt. Für fehlgeschlagene Aufträge siehe [Dashboard - Orders](/dashboard-orders).
