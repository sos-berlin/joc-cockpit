# Übersicht -  Agenten mit laufenden Jobs

Der Bereich *Agenten mit laufenden Jobs* zeigt die Anzahl der laufenden Jobs pro Agent an.

<img src="dashboard-agent-running-jobs.png" alt="Agent Running Jobs" width="350" height="140" />

## Laufende Jobs

Die Informationen sind dynamisch und können sich alle 2-3 Sekunden ändern. Die Sortierung der Agenten wird von der höchsten zur niedrigsten Anzahl von Jobs angepasst.

Der Bereich zählt nur laufende Jobs. Aufträge, die andere Arbeitsablauf-Anweisungen passieren, werden nicht gezählt. Für fehlgeschlagene Aufträge siehe [Übersicht - Aufträge](/dashboard-orders).
