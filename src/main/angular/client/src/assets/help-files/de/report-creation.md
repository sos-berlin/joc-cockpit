# Bericht erstellen

[JS7 - Reports](https://kb.sos-berlin.com/display/JS7/JS7+-+Reports) bieten Auswertungen in die zurückliegende Ausführung von Arbeitsabläufen und Jobs:

- Berichte werden aus vordefinierten [Berichtsmustern](/report-templates) erstellt, die über [Konfiguration - Inventar - Berichtsvorlagen](/configuration-inventory-reports) parametrisiert werden.
- Benutzer führen Berichte über die Ansicht *Berichtsgenerierung* oder über [JS7 - Reports - Automation](https://kb.sos-berlin.com/display/JS7/JS7+-+Reports+-+Automation) aus. Die Ansicht [Historie Berichtsläufe](/report-run-history) liefert Nachweise über zurückliegende Berichtsläufe.
- Berichte werden in der Ansicht [Berichte](/reports) visualisiert.

## Anzeige von Berichtsvorlagen

Berichte werden in der Ansicht [Konfiguration - Inventar - Berichtsvorlagen](/configuration-inventory-reports) konfiguriert.

Die Berichtsvorlagen sind verfügbar 

- über den *Navigationsbereich*, die eine Navigation zu den Berichtsvorlagen bietet.
- über den Bereich *Berichtsvorlagen*, der Details zur Berichtskonfiguration anzeigt.

### Bereich: Ordner

Der Bereich ermöglicht die Navigation durch die Ordner des Inventars, in denen die Berichtsvorlagen gespeichert sind:

- Wenn Sie einen Ordner auswählen, werden die verfügbaren Berichtsvorlagen im Bereich *Berichtsvorlagen* angezeigt.

### Bereich: Berichtsvorlage

Der Bereich zeigt die Berichtsvorlagen an, die für den aktuell ausgewählten Ordner verfügbar sind. Die Berichtsvorlagen werden mit den folgenden Eigenschaften angezeigt:

- **Name** gibt den eindeutigen Namen einer Berichtsvorlage an, wie er in der Ansicht [Konfiguration - Inventar - Berichtsvorlagen](/configuration-inventory-reports) angegeben ist.
- **Berichtsmuster** zeigt das verwendete [Berichtsmuster](/report-templates) an.
- **Häufigkeiten** teilen den *Berichtszeitraum* in gleiche Einheiten.
- **Monat von**, **Monat bis** geben Beginn und Ende des *Berichtszeitraums* an.

## Berichtslauf starten

Berichte können über das 3-Punkte Aktionsmenü einer Berichtsvorlage mit der Operation *Berichtslauf starten* erstellt werden.

Sie können eine Reihe von Berichtsvorlagen auswählen und die Massenoperation *Berichtslauf starten* anwenden, die über eine Schaltfläche in der oberen rechten Ecke des Bildschirms verfügbar ist.
