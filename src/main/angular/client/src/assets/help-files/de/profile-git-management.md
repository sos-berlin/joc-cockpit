# Profil - Git Management

Die Registerkarte *Profil - Git-Verwaltung* enthält Einstellungen für Verbindungen zu Git-Servern. Die Verbindung wird verwendet, wenn ein Bestandsordner mit einem Git-Repository verknüpft ist, um das Übertragen, Verschieben und Ziehen von Objekten zwischen der Bestandsdatenbank und dem Git-Repository zu ermöglichen.

Einzelheiten finden Sie unter [JS7 - Profiles - Git Management](https://kb.sos-berlin.com/display/JS7/JS7+-+Profiles+-+Git+Management).

<img src="profile-git-management.png" alt="Git Management" width="800" height="110" />

## Git-Verbindungen

Git-Verbindungen werden in einer Liste mit den folgenden Einträgen angezeigt:

- **Git Server** gibt den Hostnamen des Servers an, auf dem sich das Git-Repository befindet.
- **Git-Konto** ist Teil der Git-Anmeldedaten.
- **Kontoname** ist ein Teil der Git-Anmeldeinformationen
- **E-Mail-Adresse** ist die E-Mail-Adresse des Benutzerkontos.

## Operationen auf Git-Verbindungen

Die folgenden Operationen werden über Schaltflächen angeboten:

- **+** fügt eine Git-Verbindung hinzu

Die folgenden Operationen werden über das Aktionsmenü jeder Git-Verbindung angeboten:

- **Bearbeiten** bietet die Möglichkeit, die Eigenschaften der Git-Verbindung zu ändern.
- **Löschen** entfernt den Eintrag der Git-Verbindung.
