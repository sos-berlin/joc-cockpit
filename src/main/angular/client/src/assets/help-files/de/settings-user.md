# Einstellungen - Benutzer

Benutzer erhalten ein Willkommens-Pop-up-Fenster, wenn sie sich zum ersten Mal mit JOC Cockpit verbinden.

Das Willkommens-Popup-Fenster enthält Links zum Herunterladen von Standardeinstellungen für Job-Ressourcen und Beispiele für Arbeitsabläufe und verwandte Objekte.

Die Seite *Einstellungen* ist über das Symbol ![wheel icon](assets/images/wheel.png) in der Menüleiste zugänglich.

## Benutzereinstellungen

### Einstellung: *welcome\_got\_it*, Standard: *false*

Legt fest, ob die Willkommensnachricht akzeptiert wird und verhindert, dass weitere Erinnerungen erscheinen.

### Einstellung: *welcome\_do\_not\_remind\_me*, Voreinstellung: *false*

Gibt an, ob weitere Erinnerungen erfolgen sollen.
