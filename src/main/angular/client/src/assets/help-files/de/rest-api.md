# REST Web Service API

Das [JS7 - REST Web Service API](https://kb.sos-berlin.com/display/JS7/JS7+-+REST+Web+Service+API) bietet Operationen für JS7 Produkte:

- Alle Operationen, die mit Aufträgen, Arbeitsabläufen, Jobs und verwandten Objekten ausgeführt werden, wie z.B. das Abbrechen, Aussetzen und Wiederaufnehmen von Aufträgen, werden durch das *REST Web Service API* ausgeführt.
- Beachten Sie, dass jede Operation, die vom JOC Cockpit GUI angeboten wird, das *REST Web Service API* verwendet.
- Einzelheiten finden Sie in der [Technical Documentation of the REST Web Service API](https://www.sos-berlin.com/JOC/latest/raml-doc/JOC-API/index.html)
- Das [JS7 - Unix Shell Command Line Interface](https://kb.sos-berlin.com/display/JS7/JS7+-+Unix+Shell+Command+Line+Interface) wird für häufig verwendete Operationen angeboten, die auf dem *REST Web Service API* basieren.
- Darüber hinaus ist ein PowerShell Modul für den vereinfachten Zugriff auf das *REST Web Service API* verfügbar, siehe [JS7 - PowerShell Module](https://kb.sos-berlin.com/display/JS7/JS7+-+PowerShell+Module).
