# REST Web Service API

Die [JS7 - REST Web Service API](https://kb.sos-berlin.com/display/JS7/JS7+-+REST+Web+Service+API) bietet Zugriff auf Operationen mit JS7-Produkten und Zeitplan-Objekten:

- Alle Operationen, die mit Aufträgen, Arbeitsabläufen, Jobs und verwandten Objekten ausgeführt werden, wie z.B. das Stornieren, Aussetzen und Wiederaufnehmen von Aufträgen, werden von der *REST Web Service API* ausgeführt.
- Beachten Sie, dass jede Operation, die von der JOC Cockpit GUI angeboten wird, die *REST Web Service API* verwendet.
- Einzelheiten finden Sie in der [Technical Documentation of the REST Web Service API](https://www.sos-berlin.com/JOC/latest/raml-doc/JOC-API/index.html)
- Die [JS7 - Unix Shell Command Line Interface](https://kb.sos-berlin.com/display/JS7/JS7+-+Unix+Shell+Command+Line+Interface) wird für häufig verwendete Operationen angeboten, die auf der *REST Web Service API* basieren.
- Darüber hinaus ist ein PowerShell-Modul für den vereinfachten Zugriff auf die *REST Web Service API* verfügbar, siehe [JS7 - PowerShell Module](https://kb.sos-berlin.com/display/JS7/JS7+-+PowerShell+Module).
