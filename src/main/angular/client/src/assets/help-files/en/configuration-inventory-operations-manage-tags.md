# Configuración - Inventario - Operaciones - Gestionar Etiquetas

JS7 ofrece [JS7 - Tagging](https://kb.sos-berlin.com/display/JS7/JS7+-+Tagging) para Workflows, Jobs y Órdenes.

- Las Etiquetas se gestionan desde el inventario disponible en la vista *Configuración* de JOC Cockpit.
  - A los Workflows se les puede asignar cualquier número de Etiquetas.
  - A las Planificaciones se les puede asignar cualquier número de Etiquetas que se agregan a las Órdenes creadas por el [Servicio de Plan Diario](/service-daily-plan).
- Las Etiquetas pueden aplicarse al agregar Órdenes ad hoc manualmente.
- Las Etiquetas pueden usarse en la vista *Workflows*, la vista *Resumen de Órdenes*, la vista *Plan Diario* y la vista *Historial* para filtrar Workflows y Órdenes.

## Gestionar Etiquetas para un Workflow

Los usuarios pueden agregar, actualizar y eliminar Etiquetas para un único Workflow.

<img src="tags-manage-object.png" alt="Manage Tags for Workflow" width="600" height="125" />

Al ingresar Etiquetas, se ofrece la lista de Etiquetas existentes. Además, los usuarios pueden escribir el nombre de una nueva Etiqueta. Los nombres de Etiquetas distinguen entre mayúsculas y minúsculas.

## Gestionar Etiquetas para Workflows desde Carpetas

Los usuarios pueden agregar, actualizar y eliminar Etiquetas para todos los Workflows disponibles en la carpeta seleccionada y en subcarpetas de forma recursiva.

<img src="tags-manage-folder.png" alt="Manage Tags for Workflows from Folders" width="600" height="210" />

La *Lista de Etiquetas* indica el nombre de la Etiqueta y los Workflows a los que está asignada. Los usuarios pueden eliminar la asignación de una Etiqueta.
