# Profil - Permissions

La page *Profil - Permissions* indique les permissions résultantes des rôles attribués à un compte d'utilisateur.

## Rôles

Les rôles attribués au compte d'utilisateur sont affichés. Les autorisations résultantes sont fusionnées à partir des attributions de rôles.

## Permissions

### Permissions du JOC Cockpit

Les autorisations sont indiquées pour toutes les instances du JOC Cockpit.
 
### Permissions du contrôleur

Les autorisations sont indiquées par instance de contrôleur. Les autorisations *par défaut* du contrôleur s'appliquent si aucune autorisation spécifique n'est configurée pour le contrôleur concerné.
