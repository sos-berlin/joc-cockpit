# Rapports

La vue [JS7 - Reports](https://kb.sos-berlin.com/display/JS7/JS7+-+Reports) offre un aperçu de l'exécution passée des Workflows et des Tâches :

- Les rapports sont créés à partir de [Maquettes de Rapport](/report-templates) prédéfinis qui sont paramétrés à partir de [Configuration - Inventaire - Modèles de Rapport](/configuration-inventory-reports).
- Les utilisateurs exécutent les rapports à partir de la vue [Création des Rapports](/report-creation) ou de [JS7 - Reports - Automation](https://kb.sos-berlin.com/display/JS7/JS7+-+Reports+-+Automation). La vue [Historique d'Exécution des Rapports](/report-run-history) fournit des informations sur les exécutions antérieures des rapports.
- Les rapports sont visualisés dans la vue *Rapports*.

Les rapports [Historique d'Exécution des Rapports](/report-run-history) sont soumis à une purge de la base de données effectuée par le [Service d'Assainissement](/service-cleanup).

Pour l'exécution des rapports, voir [Création des Rapports](/report-creation). Pour vérifier l'exécution des rapports, voir [Historique d'Exécution des Rapports](/report-run-history).

## Liste des rapports

Les rapports affichés sont regroupés par [Maquettes de Rapport](/report-templates) et par Ordre *le plus élevé*, *le plus bas*. 

- Le développement du *Modèle de rapport* permet de sélectionner le tri *le plus élevé*, *le plus bas*.
- Le développement du tri permet d'afficher les rapports disponibles.

## Opérations sur les rapports

Les rapports peuvent être affichés en cliquant sur le rapport correspondant.

Le menu d'action à 3 points d'un rapport propose les opérations suivantes :

- **Visualiser le rapport** affiche le rapport concerné.
- **Effacer** supprime le rapport concerné.

Lorsque vous sélectionnez un certain nombre de rapports, l'opération de suppression en bloc des rapports est disponible à partir d'un bouton situé dans le coin supérieur droit de l'écran.

## Affichage des rapports

Lorsque vous cliquez sur un rapport, il s'affiche pour les *fréquences de rapport* dans la *période de rapport* donnée, comme suit :

<img src="assets/help-files/images/report-samples.png" alt="Report Samples" width="727" height="357" />

Les utilisateurs peuvent effectuer les opérations suivantes sur les rapports :

- **Filtrer par** permet de filtrer par type de fréquence, par exemple *hebdomadaire*.
- **Gérer les cartes** permet de sélectionner les *fréquences de rapport*.
- **Exporter** permet d'exporter les données du rapport vers un fichier Excel au format .xls.
