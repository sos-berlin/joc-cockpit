# Configuration - Inventaire - Opérations - Gérer Tags

JS7 propose [JS7 - Tagging](https://kb.sos-berlin.com/display/JS7/JS7+-+Tagging) pour les Workflows, les Jobs et les Ordres.

- Les étiquettes sont gérées à partir de l'inventaire disponible dans la vue *Configuration* du JOC Cockpit.
  - Un nombre quelconque de balises est attribué aux Workflows.
  - Les planifications se voient attribuer un nombre quelconque de balises qui sont ajoutées aux Ordres créés par [Daily Plan Service](/service-daily-plan).
- Les balises peuvent être appliquées lors de l'ajout manuel d'Ordres ad hoc.
- Les balises peuvent être utilisées dans les vues *Workflows*, *Ordres Overview*, *Plan Quotidien* et *Historique* pour filtrer les Workflows et les Ordres.

## Gérer les balises pour le Workflow

Les utilisateurs peuvent ajouter, mettre à jour et supprimer des balises pour un seul Workflow.

<img src="tags-manage-object.png" alt="Manage Tags for Workflow" width="600" height="125" />

Lors de la saisie des balises, la liste des balises existantes est proposée. A part cela, les utilisateurs peuvent taper le nom d'une nouvelle étiquette. Les noms d'étiquettes sont sensibles à la casse.

## Gérer les balises pour les Workflows à partir des dossiers

Les utilisateurs peuvent ajouter, mettre à jour et supprimer des balises pour tous les Workflows disponibles dans le dossier sélectionné et dans les sous-dossiers de manière récursive.

<img src="tags-manage-folder.png" alt="Manage Tags for Workflows from Folders" width="600" height="210" />

La *Liste des balises* indique le nom de la balise et les Workflows auxquels la balise est attribuée. Les utilisateurs peuvent supprimer l'attribution d'une étiquette.
