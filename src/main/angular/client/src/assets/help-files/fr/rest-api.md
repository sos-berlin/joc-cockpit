# API de service Web REST

Le site [JS7 - REST Web Service API](https://kb.sos-berlin.com/display/JS7/JS7+-+REST+Web+Service+API) permet d'effectuer des opérations sur les produits JS7 et les objets de scheduling :

- Toutes les opérations effectuées sur les Ordres, les Workflows, les Tâches et les objets connexes tels que l'annulation, la suspension et la reprise des Ordres sont effectuées par l'API *REST Web Service*.
- Notez que toute opération fournie par l'interface graphique du JOC Cockpit utilise l'API *REST Web Service*.
- Pour plus d'informations, consultez le site [Technical Documentation of the REST Web Service API](https://www.sos-berlin.com/JOC/latest/raml-doc/JOC-API/index.html)
- Le site [JS7 - Unix Shell Command Line Interface](https://kb.sos-berlin.com/display/JS7/JS7+-+Unix+Shell+Command+Line+Interface) est proposé pour les opérations fréquemment utilisées basées sur l'*REST Web Service API*.
- En outre, un module PowerShell est disponible pour un accès simplifié à l'*REST Web Service API*, voir [JS7 - PowerShell Module](https://kb.sos-berlin.com/display/JS7/JS7+-+PowerShell+Module).
