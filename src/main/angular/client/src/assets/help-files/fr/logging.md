# Logging

Le JOC Cockpit offre une fonction de journalisation pour les opérations de l'interface graphique en cas de besoin d'analyse, par exemple, des temps de réponse des [REST Web Service API](/rest-api).

Les utilisateurs peuvent spécifier les niveaux de journalisation souhaités :

- **Info**
- **stdout**
- **Avertissement**
- **Debug**

Pour activer la journalisation, la case suivante doit être cochée :

- **Log Enabled**

La sortie du journal sera disponible dans la vue *Logging*. Les paramètres de journalisation sont conservés et sont également actifs pour les séances ultérieures des utilisateurs.

## Opérations sur la sortie du journal

Lorsque la sortie du journal est disponible, les opérations suivantes peuvent être appliquées à partir des boutons connexes situés dans le coin supérieur droit de la vue :

- **Copier dans le presse-papiers** copiera la sortie du journal dans le presse-papiers de l'utilisateur.
- **Ouvrir dans un nouvel onglet** ouvrira un nouvel onglet de navigateur contenant les données de sortie du journal.
