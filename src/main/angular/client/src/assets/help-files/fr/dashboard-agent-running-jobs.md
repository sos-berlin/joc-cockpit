# Agent Running Jobs (en anglais)

L'affichage *Tâches en cours de l'Agent* indique le nombre de tâches en cours par Agent.

<img src="dashboard-agent-running-tâches.png" alt="Agent Running Jobs" width="350" height="140" />

## Tâches en cours

Les informations sont dynamiques et peuvent changer toutes les 2 ou 3 secondes. Le classement des Agents est ajusté du plus grand au plus petit nombre de tâches.

Le tableau de bord ne comptabilise que les Ordres en cours d'exécution. Les Ordres qui suivent d'autres instructions de Workflow ne sont pas comptabilisés. Pour les tâches qui ont échoué, consultez le site [Dashboard - Orders](/dashboard-orders).
