# Agent Running Jobs (en anglais)

Le panneau *Travaux en cours de l'agent* affiche le nombre de travaux en cours par agent.

<img src="dashboard-agent-running-jobs.png" alt="Agent Running Jobs" width="350" height="140" />

## Travaux en cours

Les informations sont dynamiques et peuvent changer toutes les 2 ou 3 secondes. Le classement des agents est ajusté du plus grand au plus petit nombre d'emplois.

Le tableau de bord ne comptabilise que les ordres en cours d'exécution. Les ordres qui suivent d'autres instructions de Workflow ne sont pas comptabilisés. Pour les tâches qui ont échoué, consultez le site [Dashboard - Orders](/dashboard-orders).
