# Tableau de Bord - Agents avec Tâches en exécution

L'affichage *Agents avec Tâches en exécution* indique le nombre de tâches en cours par Agent.

<img src="dashboard-agent-running-jobs.png" alt="Agents avec Tâches en exécution" width="350" height="140" />

## Tâches en exécution

Les informations sont dynamiques et peuvent changer toutes les 2 ou 3 secondes. Le classement des Agents est ajusté du plus grand au plus petit nombre de tâches.

Le Tableau de Bord ne comptabilise que les Ordres en exécution. Les Ordres qui suivent d'autres instructions de Workflow ne sont pas comptabilisés. Pour les tâches qui ont échoué, consultez [Tableau de Bord - Ordres](/dashboard-orders).
