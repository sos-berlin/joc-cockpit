# Tableau de Bord - Transfert de fichiers

La vue *Transfert de fichiers* fournit un résumé des transferts de fichiers précédents réalisés par le client YADE.

<img src="dashboard-file-transfer.png" alt="File Transfer" width="320" height="90" />

## Historique des transferts de fichiers

L'Historique récapitule les transferts réussis et échoué :

- les transferts **réussis** comprennent les tâches réussies et les tâches échouées qui se sont rétablies, par exemple après une nouvelle tentative.
- les transferts **échoués** indiquent les tâches qui ont échoué de façon permanente.

En cliquant sur le numéro indiqué, vous accédez à la vue *Historique des transferts de fichiers* qui affiche les transferts en détail.

## Filtres

Le bouton déroulant situé dans le coin supérieur droit du panneau permet de sélectionner des transferts antérieurs à partir d'une plage de dates :

- **Aujourd'hui** Les tâches sont liés à la journée en cours, qui est calculée à partir du fuseau horaire figurant dans le profil de l'utilisateur.
- **Dernière heure** inclut les tâches effectuées au cours de la dernière heure.
- **Les 12 dernières heures** comprennent les tâches qui ont été terminés au cours des 12 dernières heures.
- **Les dernières 24 heures** comprennent les tâches terminés au cours des dernières 24 heures.
- **Les 7 derniers jours** comprennent les tâches qui ont été terminés au cours des 7 derniers jours.
