# Filtre avancé

Les filtres avancés permettent de limiter les résultats de la recherche dans un certain nombre de vues.

## Personnalisation

- Les utilisateurs peuvent spécifier le nom d'une personnalisation dans laquelle le filtre sera stocké en vue d'une réutilisation ultérieure.
- Les personnalisations sont disponibles dans la plupart des vues et peuvent être activées à partir d'un bouton déroulant situé sous le menu principal.

## Critères de recherche

Les critères de recherche sont cumulatifs et ajoutent aux résultats toutes les entrées correspondant à l’ensemble des critères spécifiés.

La recherche dans les champs est insensible à la casse et partielle, par exemple :

- **rest** trouvera les objets portant le nom "pdfNon**Rest**artable" et "**REST**-RunningTaskLog"
- **re?t** trouvera les objets portant le nom "ActivePassiveDi**rect**or" et "JITL-JS7**REST**ClientJob"
- **re?t** trouvera les objets portant le nom "pdSQLExecuto**rExt**ractJSON" et "pdu**Reset**Subagent"

### Résultats de la recherche

Les résultats de la recherche sont proposés pour les opérations en masse telles qu'elles sont disponibles sur la page correspondante.
