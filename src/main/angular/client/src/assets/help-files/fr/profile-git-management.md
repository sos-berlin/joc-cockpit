# Profil - Gestion du Git

L'onglet *Profil - Gestion Git* contient les paramètres pour les connexions aux serveurs Git. La connexion est utilisée si un dossier d'inventaire est associé à un dépôt Git pour permettre la transmission, le transfert et l'extraction d'objets entre la base de données d'inventaire et le dépôt Git.

Pour plus de détails, voir [JS7 - Profiles - Git Management](https://kb.sos-berlin.com/display/JS7/JS7+-+Profiles+-+Git+Management).

<img src="profile-git-management.png" alt="Git Management" width="800" height="110" />

## Connexions Git

Les connexions Git sont affichées à partir d'une liste contenant les entrées suivantes :

- **Git Server** indique le nom d'hôte du serveur qui contient le dépôt Git.
- **Compte Git** fait partie des informations d'identification Git.
- **Nom du compte** fait partie des informations d'identification Git
- **L'adresse électronique** est l'adresse électronique du compte d'utilisateur.

## Opérations sur les connexions Git

Les opérations suivantes sont proposées à partir des boutons :

- **+** ajoute une connexion Git

Les opérations suivantes sont proposées dans le menu d'action de chaque connexion Git :

- **Editer** permet de modifier les propriétés de la connexion Git.
- **Supprimer** permet de supprimer l'entrée de la connexion Git.
