# ダッシュボード - オーダー

*オーダー* ウイジェットには、以下のオーダーに関する情報が表示されます：

- [実行計画](/daily-plan) によって追加されたオーダー。
- [ワークフロー](/workflows) 画面からユーザーがオンデマンドで追加したオーダー。
- [ファイルオーダーソース](/configuration-inventory-file-order-sources) から追加されたオーダー。

<img src="dashboard-orders.png" alt="Orders" width="330" height="140" />

## オーダーステータス

*オーダー* ウイジェットは、オーダーの現在の状態に関する情報を提供します。このパネルは、オーダーの状態が変更されると更新されます。

- **保留中** オーダーは開始時刻を指定せずにワークフローに追加されています。
- **予定** オーダーは開始時刻を設定してワークフローに追加され、後日実行される予定です。
- **進行中** オーダーはワークフローを処理していますが、ジョブは実行されていません。 
- **実行中** オーダーはジョブ実行中です。 
- **一時停止** オーダーはユーザーの介入により中断され、再開することができます。
- **完了** オーダーはワークフローを完了しましたが、削除されていません。例えば、ファイルオーダーソースがファイル監視に使用され、ワークフローが受信ファイルを（再）移動しない場合などです。この場合、ファイルが受信ディレクトリに存在する限り、オーダーはそのまま残ります。
- **指示待ち** オーダーは、ワークフローの *Prompt命令* により保留され、ワークフローの実行を続行するためにユーザーの確認が必要です。
- **待機中** オーダーは、*リソースロック*、*通知*、*リトライ*、または *サイクル* インターバルなどのリソースを待機します。
- **ブロック中** オーダーは、オーダーが追加されてからエージェントに到達できない場合などに開始できません。
- **失敗** オーダーは、ジョブが失敗したか、*Fail命令*によってオーダーが実行できないことを示します。 

表示された数のオーダーをクリックすると、[オーダー](/orders-overview) 画面に移動し、オーダーの詳細が表示されます。

## 検索ボタン

パネルの右上にあるドロップダウンボタンは、日付範囲からオーダーを選択することができます：

- **全て** は、コントローラーとエージェントで利用可能なすべてのオーダーを表示します。
- **本日** は、ユーザーの[プロファイル - 設定値](/profile-preferences) のタイムゾーンから計算される現在の日のオーダーが表示されます。
  - **保留** は、開始時刻を指定せずにワークフローに追加されたオーダーです。これらのオーダーは後で開始時刻を設定できます。
  - **予定** は、開始時刻を指定してワークフローに追加されたオーダーです。指定された日時に実行される予定です。
  - **進行中** は、実行中のオーダーですが、現在ジョブは実行されていません、
  - **実行中** は、ジョブを実行中のオーダーです。
  - **一時停止** は、ユーザー操作によって停止されているオーダーで、再開可能です。
  - **完了** は、ワークフローを完了しましたが、削除されていないオーダーです。例えば、ファイル監視用にファイルオーダーソースを使用している場合、ワークフローが受信ファイルを（再）移動しないケースがあります。この場合、ファイルが受信ディレクトリに存在している限り、注文はそのままの状態に保持されます。
  - **指示待ち** は、ワークフロー内の *Prompt命令* によって一時的に保留状態となり、ワークフローの実行を継続するにはユーザーによる確認が必要です。
  - **待機中** のオーダーは、リソースロック、通知、再試行、サイクル間隔などのリソースが利用可能になるのを待つか、使用中のエージェントが設定したプロセス制限を超えた場合に処理が待機状態になります。
  - **ブロック中** のオーダーは、オーダーが追加された時点で使用するエージェントが使用できない場合などにオーダーを開始できません。
  - **失敗** のオーダーは、ジョブが失敗したか、*Failed命令* が設定されているためにオーダーが進行できない状態であることを示します。
- **今から1時間**には、次の1時間に *計画* されたオーダーが含まれます。
- **今から12時間**には、次の12時間に *計画* されたオーダーが含まれます。
- **今から24時間**には、次の24時間に *計画* されたオーダーが含まれます。
- **翌日**には、翌日末までに *計画* されたオーダーが含まれます。
- **今から7日間**には、翌7日後までに *計画* されたオーダーが含まれます。
