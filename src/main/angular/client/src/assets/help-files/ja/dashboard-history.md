# 履歴

履歴(*History*)パネルは過去のオーダーとジョブの実行に関する情報を提供します。

<img src="dashboard-history.png" alt="History" width="330" height="80" />

## 履歴ステータス

履歴ステータスはオーダーやジョブが完了したときの最終ステータスです。履歴ステータスは進行中のオーダーやジョブを考慮しません。表示されたオーダーやジョブは履歴であり、操作はできません。

- 成功したオーダー(**Successful Orders**)は、成功した結果で完了します。これには、自動エラー処理またはユーザーの操作によって回復した、失敗したオーダーも含まれます。
- **失敗したオーダー**は、失敗したジョブや*失敗指示*などの問題が発生したものです。
- **成功したジョブ**は、成功した結果で完了します。これには、関連するオーダーが完了していないワークフロー内のジョブも含まれます。
- **失敗したジョブ**は、ジョブの実行時に問題が発生しました。

指定された数のオーダーまたはジョブをクリックすると、*Order History*ビューまたは*Task History*ビューに移動し、オーダーとジョブの詳細が表示されます。

## フィルター

パネルの右上にあるドロップダウンボタンは、過去のオーダーやジョブを日付の範囲から選択することができます：

- **今日** オーダーとジョブは、ユーザープロフィールのタイムゾーンから計算される現在の日に関連しています。
- **直近1時間**のオーダーとジョブは、直近1時間に完了したオーダーとジョブです。
- **直近12時間以内**には、直近12時間以内に完了したオーダーとジョブが含まれます。
- **直近24時間以内**には、直近24時間以内に完了したオーダーとジョブが含まれます。
- **直近7日間**には、直近7日間に完了したオーダーとジョブが含まれます。
