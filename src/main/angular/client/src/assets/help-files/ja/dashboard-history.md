# ダッシュボード - 履歴

*履歴* ウィジェットは過去のオーダーとジョブの実行に関する履歴を提供します。

<img src="dashboard-history.png" alt="History" width="330" height="80" />

## 履歴ステータス

履歴ステータスはオーダーやジョブが完了したときの最終ステータスです。履歴ステータスは進行中のオーダーやジョブを考慮しません。表示されたオーダーやジョブは履歴であり、操作はできません。

- **完了オーダー** は、成功した結果で完了したオーダー数です。これには、自動エラー処理またはユーザーの操作によって回復した、失敗したオーダーも含まれます。
- **失敗オーダー** は、失敗したジョブや *Failed命令* などの問題が発生したものです。
- **成功ジョブ** は、成功した結果で完了したジョブ数です。これには、オーダーが完了していないワークフロー内のジョブも含まれます。
- **失敗ジョブ** は、ジョブの実行時に問題が発生した数です。

指定された数のオーダーまたはジョブをクリックすると、*オーダー履歴* 画面、または *タスク履歴* 画面に移動し、オーダーとジョブの詳細が表示されます。

## 検索ボタン

パネルの右上にあるドロップダウンボタンは、過去のオーダーやジョブを日付の範囲から選択することができます：

- **本日** のオーダーとジョブは、ユーザープロファイルのタイムゾーンから計算される現在の日の履歴を表示します。
- **最新1時間** は、直近1時間に完了したオーダーとジョブです。
- **最新12時間** は、直近12時間以内に完了したオーダーとジョブが含まれます。
- **最新24時間** は、直近24時間以内に完了したオーダーとジョブが含まれます。
- **最新7日間** は、直近7日間に完了したオーダーとジョブが含まれます。
