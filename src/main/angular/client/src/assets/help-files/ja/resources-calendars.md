# リソース - カレンダー

*カレンダー* 画面では、カレンダーの使用状況が表示されます。

- **営業日カレンダー** は、*実行計画* からオーダーを作成する日を指定します。
- **休業日カレンダー** は、オーダーを作成しない日を指定します。

カレンダーは、実行計画がオーダーを作成するために使用するスケジュールによって参照されます。

## ナビゲーションパネル

左側のパネルにはカレンダーを保持するインベントリフォルダーのツリーが表示されます。

- フォルダーをクリックすると、そのフォルダーのカレンダーが表示されます。
- フォルダーにマウスカーソルを合わせると表示される三角アイコンをクリックすると、そのフォルダーおよびすべてのサブフォルダーのカレンダーが表示されます。

検索アイコンをクリックすると、ユーザー入力に基づいてカレンダーを検索できます：

- Test**と入力すると、*test-calendar-1* や *TEST-calendar-2* といった名前のカレンダーが表示されます。 
- Test**と入力すると、*test-calendar-1* や *my-TEST-calendar-2* といった名前のカレンダーが表示されます。

## カレンダーパネル

### カレンダーの表示

以下の情報が表示されます：

- **Name**はカレンダーの固有名です。
- **タイプ**は、*営業日カレンダー*または*非営業日カレンダー*のいずれかです。
- **有効期限**、**有効期限**は、オプションで有効期間を示します。有効期間のないカレンダーは無制限に有効です。

### カレンダーの操作

以下の操作が可能です：

- **プレビューを表示** カレンダーから返された日付を表示します。
