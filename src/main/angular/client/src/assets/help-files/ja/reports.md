# レポート

[JS7 - Reports](https://kb.sos-berlin.com/display/JS7/JS7+-+Reports) は過去のワークフローとジョブの実行から得られた洞察を提供します：

- レポートは、[Configuration - Inventory - Reports](/configuration-inventory-reports) からパラメータ化された、事前に定義された[Report Templates](/report-templates) から作成されます。
- ユーザーは[Report - Creation](/report-creation) ビューまたは[JS7 - Reports - Automation](https://kb.sos-berlin.com/display/JS7/JS7+-+Reports+-+Automation) からレポートを実行します。[Report - Run History](/report-run-history) ビューは、過去のレポート実行に関する証拠を提供します。
- レポートは *Reports* ビューで視覚化されます。

レポートおよび[Report - Run History](/report-run-history) は、[Cleanup Service](/service-cleanup) によって実行されるデータベースのパージの対象となります。

レポートの実行については、[Report - Creation](/report-creation) を参照してください。レポート実行の確認については、[Report - Run History](/report-run-history) を参照してください。

## レポートのリスト

表示されるレポートは[Report Templates](/report-templates) および *最高*、*最低* のソートでグループ化されています。 

- レポート テンプレート]を展開すると、[最高]、[最低]の並べ替えを選択できます。
- 並べ替えを展開すると、利用可能なレポートが表示されます。

## レポートの操作

関連するレポートをクリックすると、レポートを表示できます。

レポートの3点アクションメニューでは以下の操作が可能です：

- **レポートを表示** 関連レポートを表示します。
- **削除** 関連レポートを削除します。

複数のレポートを選択すると、画面右上のボタンからレポートを一括削除できます。

## レポートの表示

レポートをクリックすると、以下のようにレポート期間中の*レポート頻度*が表示されます：

<img src="assets/help-files/images/report-samples.png" alt="Report Samples" width="727" height="357" />

ユーザーはレポートに対して以下の操作を行うことができます：

- **フィルタ・バイ**では、*weekly*のような頻度タイプによるフィルタリングが可能です。
- **カードの管理**では、*レポートの頻度*を選択できます。
- **エクスポート**では、レポートデータを.xls形式のExcelファイルにエクスポートできます。
