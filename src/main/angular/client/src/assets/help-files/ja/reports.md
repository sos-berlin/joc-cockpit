# レポート - レポート

[JS7 - Reports](https://kb.sos-berlin.com/display/JS7/JS7+-+Reports) は過去のワークフロー及びジョブ実行結果のレポートを提供します：

- レポートは、[ジョブ定義 - インベントリー - レポート](/configuration-inventory-reports) から事前に定義した[レポートテンプレート](/report-templates) から作成されます。
- ユーザーは[レポート - 新規レポート](/report-creation) 画面または[JS7 - Reports - Automation](https://kb.sos-berlin.com/display/JS7/JS7+-+Reports+-+Automation) からレポートを作成します。[レポート - 実行履歴](/report-run-history) 画面は、過去のレポート実行履歴を表示します。
- レポートは *レポート* 画面で表示されます。

レポートおよび[レポート - 実行履歴](/report-run-history) は、[クリーンアップサービス](/service-cleanup) によるデータベースの削除の対象となります。

レポートの実行については、[レポート - 作成](/report-creation) を参照してください。レポート実行の確認については、[レポート - 実行履歴](/report-run-history) を参照してください。

## レポート

表示されるレポートは[レポートテンプレート](/report-templates) および *降順*、*昇順* でグループ化されています。

- *レポート テンプレート* を展開すると、[降順]、[照準]の並べ替えを選択できます。
- 並べ替えを展開すると、利用可能なレポートが表示されます。

## レポートの操作

関連するレポートをクリックすると、レポートを表示できます。

レポートのアクションメニューでは以下の操作が可能です：

- **レポート表示** 関連レポートを表示します。
- **削除** 関連レポートを削除します。

複数のレポートを選択すると、画面右上のボタンからレポートを一括削除できます。

## レポートの表示

レポートをクリックすると、以下のようにレポート期間中の*レポート実行頻度*が表示されます：

<img src="assets/help-files/images/report-samples.png" alt="Report Samples" width="727" height="357" />

ユーザーはレポートに対して以下の操作を行うことができます：

- **フィルター**では、*毎週*のような頻度タイプによるフィルタリングが可能です。
- **カード管理**では、*レポートの頻度*を選択できます。
- **エキスポート**では、レポートデータを.xls形式のExcelファイルにエキスポートできます。
