#エージェントが実行中のジョブ

Agent Running Jobs* パネルは、エージェントごとに実行中のジョブ数を表示します。

<img src="dashboard-agent-running-jobs.png" alt="Agent Running Jobs" width="350" height="140" />

## 実行中のジョブ

この情報は動的であり、2-3秒ごとに変化します。エージェントのソートは、ジョブ数の多い順から少ない順に調整されます。

パネルは実行中のジョブのみをカウントし、他のワークフロー指示を通過したオーダーはカウントされません。失敗したジョブについては[Dashboard - Orders](/dashboard-orders) を参照してください。
