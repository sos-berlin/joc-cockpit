# ダッシュボード - ジョブ実行中エージェント

*ジョブ実行中エージェント* ウィジェットは、エージェントごとに実行中のジョブ数を表示します。

<img src="dashboard-agent-running-jobs.png" alt="Agent Running Jobs" width="350" height="140" />

## 実行中ジョブ

この情報は動的であり、2-3秒ごとに更新されます。エージェントの表示順は、ジョブ数の多い順から少ない順に調整されます。

ウィジェットは実行中のジョブのみをカウントし、他のワークフロー命令を実行したオーダーはカウントされません。失敗したジョブについては[ダッシュボード - オーダー](/dashboard-orders) を参照してください。
