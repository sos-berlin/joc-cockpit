# モニター - エージェントモニター

この画面には、エージェントインスタンスの稼働率のインジケータが表示されます。

エージェントクラスターが使用されている場合、クラスター全体の稼働率が考慮されます。例えば、クラスター内のエージェントがメンテナンスのためにシャットダウンされ、残りのインスタンスが負荷を補う場合、稼働率は低下しません。

画面右上にある、*現在のコントローラー* チェックボックスのチェックを外すと、接続されているすべてのコントローラーのエージェントの稼働率が表示され、チェックを入れると、現在選択されているコントローラーのみに登録されているエージェントの情報が表示されます。

[クリーンアップサービス](/service-cleanup) によって、エージェントの稼働率の履歴データは消去される可能性があることに注意してください。

## 日付フィルタ

パネルの右上隅では、日付範囲を選択することができます：

- **週** は、日付スライダーを1週間単位に切り替えます。
- **月** は、1ヶ月単位でスライダーを切り替えます。
- **カスタム** は、開始日と終了日を指定します。

## 実行時間

指定された期間にエージェントが利用可能であると確認された割合を示します。

## 統計情報

指定された期間内の稼働状況を日毎に棒グラフで表示します。各エージェントは1日ごとに表示されます。 

## 概要

指定された期間のエージェントごと、日ごとの稼働率を表示します。

- チャートは、エージェントの可用性が確認された時間帯を緑色で表示します。 
- 赤色は利用不可を示します。
- 灰色はデータの欠落を示します。
