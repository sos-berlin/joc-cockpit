# 設定 - User

JOCコックピットに初めてログインする際、ユーザーにはウェルカムポップアップウィンドウが表示されます。

ウェルカムポップアップウィンドウには、ジョブリソースのデフォルト、ワークフローと関連オブジェクトのサンプルをダウンロードするためのリンクがあります。

メニューバーの ホイールアイコンから *設定* ページにアクセスできます。

## ユーザー設定

### 設定 *welcome_got_it*, Default： *false*。

ウェルカムメッセージが承認済みであるかどうかを指定し、それ以降のリマインダーが発生しないようします。

### 設定： *welcome_do_not_remind_me*, Default： *false

それ以降もリマインダーを発生させるかどうかを指定します。
