# 設定 - ユーザー

JOC Cockpitに初めて接続する際、ユーザーにはウェルカムポップアップウィンドウが表示されます。

ウェルカムポップアップウィンドウには、ジョブリソースのデフォルト、ワークフローと関連オブジェクトのサンプルをダウンロードするためのリンクがあります。

メニューバーの ![wheel icon](assets/images/wheel.png) アイコンから *設定* ページにアクセスできます。

## ユーザー設定

### 設定 *welcome_got_it*, Default： *false*。

ウェルカムメッセージが受理され、さらにリマインダーが発生しないようにするかどうかを指定します。

### 設定： *welcome_do_not_remind_me*, Default： *false

さらにリマインダーを発生させるかどうかを指定します。
