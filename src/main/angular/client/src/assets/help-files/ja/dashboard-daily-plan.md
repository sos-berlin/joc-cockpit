# ダッシュボード - 実行計画

*実行計画* ウィジェットは、実行計画によって作成されたオーダーの情報が表示されます。ユーザーの操作によりオンデマンドで作成されたオーダーや、*ファイルオーダーソース* により作成されたオーダーは除外されます。

<img src="dashboard-daily-plan.png" alt="Daily Plan" width="330" height="80" />

## 実行計画ステータス

実行計画ステータスは、オーダーがデイリープランサービスによって作成されたときの初期ステータスです。

- **開始予定** オーダーはコントローラーおよびエージェントに登録されていません。*開始予定* オーダーの数は、日付範囲がオーダーを送信すべき日数の範囲内にある場合に問題があることを示します。
- **登録済** オーダーは、その日のうちに実行される予定であるか、実行中です。このステータスは、開始されたオーダーの現在の状態を反映するものではなく、日中に実行されるはずのオーダーを要約したものです。
- **完了** オーダーは完了しました。これは、*オーダー履歴* 画面で表示される、オーダーが正常に完了したか失敗したかの結果とは無関係です。

表示されているオーダー数をクリックすると、*実行計画*　画面に移動し、オーダーの詳細が表示されます。

## 検索ボタン

パネルの右上にあるドロップダウンボタンで、日付範囲からオーダーを選択することができます：

- **本日** は、ユーザープロファイルのタイムゾーンから計算される現在の日のオーダーを表示します。
- **翌日** は、翌日のオーダーを対象としています。ただし、*本日* のオーダーは除きます。
- **翌々日** は、翌々日のオーダーを対象とします。
- **翌3日** は、翌3日目のオーダーを対象とします。
- **翌4日** は、翌4日目のオーダーを対象とします。
- **翌5日** は、翌5日目のオーダーを対象とします。
- **翌6日** は、翌6日目のオーダーを対象とします。
- **翌7日** は、翌7日目のオーダーを対象とします。
