# REST ウェブサービス API

[JS7 - REST Web Service API](https://kb.sos-berlin.com/display/JS7/JS7+-+REST+Web+Service+API) は、JS7 の製品とスケジュールオブジェクトに対する操作へのアクセスを提供します：

- オーダー、ワークフロー、ジョブ、およびオーダーのキャンセル、中断、再開などの関連オブジェクトに対して実行されるすべての操作は、*REST Web Service API*によって実行されます。
- JOCコックピットGUIで提供される操作は全て*REST Web Service API*を使用します。
- 詳細は[Technical Documentation of the REST Web Service API](https://www.sos-berlin.com/JOC/latest/raml-doc/JOC-API/index.html)
- REST Web Service API* に基づいて頻繁に使用される操作については、[JS7 - Unix Shell Command Line Interface](https://kb.sos-berlin.com/display/JS7/JS7+-+Unix+Shell+Command+Line+Interface) 。
- さらに、*REST Web Service API* へのアクセスを簡略化するための PowerShell モジュールも用意されています（[JS7 - PowerShell Module](https://kb.sos-berlin.com/display/JS7/JS7+-+PowerShell+Module) を参照）。
