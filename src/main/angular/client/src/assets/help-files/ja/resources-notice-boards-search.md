# 通知ボード検索

*通知ボード検索*は、以下のような条件に基づいて通知ボードを検索するために使用されます。

- 指定された名前またはタイトルにマッチし、オプションでフォルダによって制限されます。

## メタ文字

- **?**メタ文字は任意の1文字を置き換えます。
- **\*** メタ文字は0文字以上を置き換えます。

検索は大文字と小文字を区別せず、部分的に修飾して行われます。

- **test**は "My-**Test**-Board-1 "と "**TEST**-Board-2 "という名前の通知ボードを検索します。
- **te?t**は "Global-**Test**-Board-1 "と "**TEXT**-Board-2 "という名前の通知ボードを検索します。
- **te\*t**は、"My-**tExt***-Board-1 "と "My-**Terminat***ing-Board-2 "という名前の通知ボードを検索します。

## 詳細検索

この機能は<br/>**&gt; 詳細** をクリックすると利用できます。

### 属性検索

詳細検索では、オブジェクトの属性で検索することができます：

- **エージェント名**は、指定されたエージェントで実行されたジョブを含むワークフローの通知ボードを返します。
- **ジョブ数** は、**From** 条件で指定された最小数のジョブを使用するワークフローの通知ボードを返します。**To**条件と併用した場合、*From*から*To*の範囲内のジョブ数を含むワークフローが返されます。**To**条件のみを使用した場合、*To*条件を超えるジョブ数を含まないワークフローが返されます。
- **ジョブ名**は、指定された名前に一致するジョブが含まれるワークフローの通知ボードを返します。**ジョブ名完全一致**チェックボックスを使用する場合、入力された検索語は大文字と小文字の区別を含むジョブ名と完全に一致する必要があります。

### 依存関係検索

検索メタ文字 **\*** は、例えばロックがどのような名前であっても、依存関係が検索されることを指定するために使用されます：

- **ロック** の **\*** メタ文字は、ロックを使用するワークフローの通知ボードを返します、
- **ファイルオーダーソース** の **\*** メタ文字は、ファイルオーダーソースによって参照されるワークフローの通知ボードを返します。
