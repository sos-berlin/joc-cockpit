# プロフィール - Git管理

プロファイル - Git 管理* タブには Git サーバーへの接続設定があります。この接続は、インベントリフォルダが Git リポジトリに関連付けられている場合に使用され、インベントリデータベースと Git リポジトリの間でオブジェクトのコミットやプッシュ、プルなどができるようになります。

詳細は[JS7 - Profiles - Git Management](https://kb.sos-berlin.com/display/JS7/JS7+-+Profiles+-+Git+Management) を参照してください。

<img src="profile-git-management.png" alt="Git Management" width="800" height="110" />

## Git の接続

Git接続は、以下のエントリーを持つリストから表示されます：

- **Git Server** は、Git リポジトリを保持しているサーバーのホスト名を表します。
- **Git Account** は Git の認証情報の一部です。
- **アカウント名** は Git 認証情報の一部です。
- **E-Mail Address** はユーザーアカウントのメールアドレスです。

## Git コネクションの操作

以下の操作はボタンから行います：

- **+** Git 接続の追加

各 Git 接続のアクションメニューには、次のような操作があります：

- **Edit**はGit接続のプロパティを変更します。
- **Delete**はGit接続のエントリーを削除します。
