# プロファイル - Git管理

*プロファイル - Git 管理* タブには Gitサーバーへの接続設定があります。この接続は、インベントリーフォルダが Gitリポジトリに関連付けられている場合に使用され、インベントリーデータベースと Gitリポジトリの間でオブジェクトのコミットやプッシュ、プルなどができるようになります。

詳細は[JS7 - Profiles - Git Management](https://kb.sos-berlin.com/display/JS7/JS7+-+Profiles+-+Git+Management) を参照してください。

<img src="profile-git-management.png" alt="Git Management" width="800" height="110" />

## Gitの接続

Git接続は、以下の項目を持つリストから表示されます：

- **Gitサーバー** は、Git リポジトリサーバーのホスト名を表します。
- **Gitアカウント** は、Git リポジトリに登録している認証情報の一部です。
- **ユーザー名** は、 Git リポジトリに登録しているユーザー名です。
- **メールアドレス** は、ユーザーアカウントのメールアドレスです。

## Git 接続の操作

以下の操作はボタンから行います：

- **+** Git 接続の追加

各 Git 接続のアクションメニューには、次のような操作があります：

- **編集** は、Git接続のプロパティを変更します。
- **削除** は、Git接続のエントリーを削除します。
