# ダッシュボード - エージェントステータス

*エージェントステータス* ウィジェットは、登録されたエージェントの接続ステータスを表示します。

<img src="dashboard-agent-status.png" alt="Agent Status" width="332" height="135" />

## アーキテクチャー

### エージェント

- **スタンドアロンエージェント**は、オンプレミスとコンテナからリモートマシン上でジョブを実行します。エージェントは個別に運用され、コントローラによって管理されます。
- **エージェントクラスター**
  - **ディレクターエージェント**は、エージェントクラスター内の　*サブエージェント* を管理します。これらのエージェントはアクティブパッシブクラスタリングで2つのインスタンスから構成され、コントローラーによって管理されます。
  - **サブエージェント**は、オンプレミスやVMのリモートマシンやコンテナからジョブを実行します。これらはエージェントクラスターのワーカーノードと考えることができ、**ディレクターエージェント** によって管理されます。

### 接続

- **スタンドアロンエージェント**、**ディレクターエージェント** への接続は、コントローラーから接続されます。 
- エージェントクラスター内の **サブエージェント** への接続は、**ディレクターエージェント** から接続されます。

## 接続ステータス

エージェントのステータス表示には、以下の色表示が使用されます：

- **緑色** は、正常なエージェント接続を示します。
- **黄色** は、現在リセット中のエージェントを示します。
- **赤色** は、エージェントへの接続に失敗したことを示します。
- **灰色** は、接続ステータスが *不明* であることを示します。例えば、ディレクターエージェントに接続できない場合、その配下のサブエージェントのステータスは *不明* です。

ユーザーは以下の影響を考慮する必要があります：

- エージェント接続が失敗とみなされた場合、エージェントがダウンしていることを示すものではありません。接続を妨げるネットワークの問題がある可能性があります。
- JOC コックピットは、コントローラーからエージェント接続ステータスの情報を受け取ります。コントローラーが利用できない場合、この情報は表示されません。これはエージェントがダウンしていることを意味しませんが、エージェントは *不明* ステータスから表示されることを意味します。
- コントローラーは、*スタンドアロンエージェント* と *ディレクターエージェント* への接続をJOCコックピットに報告します。*ディレクターエージェント* への接続に失敗した場合、コントローラーはエージェントクラスター内の *サブエージェント* のステータスを把握していないため、*不明* と表示されます。
