# 操作ログ

*操作ログ* は、スケジュールオブジェクトの変更を記録します。

## リクエスト

JOCコックピットでは、すべてのユーザー操作は REST API リクエストから実行されます。

リクエストは使用されるエンドポイントを識別し、三角アイコンをクリックするとリクエストボディを表示します。

## カテゴリー

- **コントローラー** は、オンデマンドでオーダーを追加するようなコントローラーの操作を示します。
- **実行計画** は、実行計画の変更を示します。
- **IDサービス** は、IDサービスの変更を示します。
- **インベントリー** は、ワークフローの保存など、インベントリの変更を示します。

# 監査証跡

JOCコックピットは、*操作ログ* とは別に REST API リクエストを追跡する *監査証跡* ファイルを書き込むように構成できます。*監査証跡* にはユーザーセッションに関する情報が含まれ、JOCコックピットGUI画面からは利用できません。

この *監査証跡* は管理者が有効にすることができ、ディスク上のログファイルから利用できます。
