# ジョブ定義 - インベントリー - 実行計画 - レポート

*レポートパネル* では、ワークフローやジョブの実行に関するレポートを設定することができます：

- レポートの構成はJOCコックピット *ジョブ定義* 画面で利用可能なインベントリーから管理されます。
  - **テンプレート** は、失敗したワークフローの上位 10 件、失敗したジョブの上位 100 件などよく使われるレポートのテンプレートです。完全なリストについては、[レポートテンプレート](/report-templates) を参照してください。
  - **間隔** は、レポートを集計する日付範囲です。日付範囲は絶対または相対のどちらでもかまいません。
  - **周期** は、*レポート期間* を同じ週または月に分割します。
- レポートの実行と表示は、JOCコックピットの *レポート* 画面から利用できます。

レポートは以下のパネルから管理できます：

- ウィンドウの左側にある[ジョブ定義 - インベントリー - ナビゲーションパネル](/configuration-inventory-navigation) から、レポートのフォルダーによるナビゲーションを提供します。また、このパネルではレポートに対する操作を行うことができます。
- ウィンドウ右側の *レポートパネル* はレポート設定の詳細を保持します。

## レポートパネル

レポートでは以下の入力が可能です：

- **名称** は、レポートの一意の識別子です。[オブジェクト命名規則](/object-naming-rules) を参照してください。
- **タイトル** はレポートの目的などを説明します。 
- **テンプレート** は、使用する[レポートテンプレート](/report-templates) を選択します。
- **間隔** は日付範囲を指定します：
  - **レポート期間**
    - *レポート期間* の開始月と終了月の月数を指定します。例えば、先月は *1m* から *1m* までです。
  - **相対**
    - *単位* は *年* 、*月*、*四半期* のいずれかです。
    - *自*は、*レポート期間* の開始単位となる過去の単位数を指定します。
    - *カウント*は、*間隔*が終了する過去の単位数を指定します。
  - **プリセット** は、*先月*、*今四半期*、*直近四半期*、*今年*、*昨年* のような定義済みの日付範囲の数から選択することができます。
- **ソート**
  - **最大**： レポートは最高値の上位n個の値を返します。
  - **最小**：このレポートは、上位 n 個の最低値を返します。
- **周期**は、レポート期間を同じ時間単位に分割します：
  - *毎週*
  - *隔週*
  - *毎月*
  - *3ヶ月毎*
  - *半年毎*
  - *毎年*
  - *3年毎*

## レポートの運用

一般的な操作については[ジョブ定義 - インバントリー - ナビゲーションパネル](/configuration-inventory-navigation) を参照してください。

レポートに関する操作は、以下のビューから実行できます：

- レポートは[レポート - 新規レポート](/report-creation) 画面から作成します。
- レポートの実行は、[レポート - レポート実行](/report-run-history) 画面から利用できます。
- レポートは、[レポート](/reports)画面から視覚化できます。
