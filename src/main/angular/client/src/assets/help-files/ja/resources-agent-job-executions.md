# エージェントのジョブ実行

Agent Job Executions* ビューは、指定された期間におけるエージェントのジョブ実行を要約します。

エージェントには以下の種類があります：

- **スタンドアロンエージェント** は、オンプレミスのリモートマシンやコンテナからジョブを実行します。エージェントは個別に操作され、コントローラによって管理されます。
- **エージェントクラスタ
  - Directorエージェントは、エージェントクラスタ内の**Subagents**をオーケストレーションします。さらに、ジョブの実行にも使用できます。
  - **Subagents**はオンプレミスのリモートマシンやコンテナからジョブを実行します。これらはエージェントクラスタのワーカーノードと考えることができ、**Director Agent**によって管理されます。

## エージェントのジョブ実行パネル

以下の情報が表示されます：

- **エージェント名** はエージェントの固有名です。
- **URL** はコントローラからエージェントにアクセスできる URL です。
- **Number of successfully executed tasks** はタイトルが示す通りです。
- **Number of tasks executed** には、成功したジョブ実行と失敗したジョブ実行が含まれます。

## エージェントのジョブ実行のエクスポート

ユーザは表示されたサマリー情報を .xlsx 形式の Excel ファイルにエクスポートできます。エクスポート時にはアクティブなフィルタとソートオーダーが適用されます。

## フィルタ

ユーザーはウィンドウの上部にあるフィルターを適用して、ジョブ実行のサマリーを制限することができます：

- **Date Range** フィルターボタンはジョブ実行のサマリーの日付範囲を選択します。
- **Current Controller** チェックボックスは、ジョブの実行を現在選択されているコントローラに 登録されているエージェントに限定します。

## 高度なフィルター

アドバンストフィルタ]は、より広い日付範囲、特定のエージェントインスタンス、コントローラインスタンスのフィルタ条件を提供します。

アドバンストフィルタ(*Advanced Filter*) は、指定された条件でデータをエクスポートすることができます。
