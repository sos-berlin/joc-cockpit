# リソース - エージェント実行ジョブ

*エージェント実行ジョブ* 画面は、指定された期間におけるエージェントのジョブ実行を表示します。

エージェントには以下の種類があります：

- **スタンドアロンエージェント** は、オンプレミスのリモートマシンやコンテナからジョブを実行します。エージェントは個別に操作され、コントローラによって管理されます。
- **エージェントクラスター**
  - **ディレクターエージェント** は、エージェントクラスター内の **サブエージェント** を管理します。また、ジョブ実行にも使用できます。
  - **サブエージェント** は、オンプレミスのリモートマシンやコンテナからジョブを実行します。これらはエージェントクラスタのワーカーノードと考えることができ、**Director Agent** によって管理されます。

## エージェントのジョブ実行画面

以下の情報が表示されます：

- **エージェント名** は一意なエージェントの名称です。
- **URL** はコントローラーからエージェントにアクセスできる URL です。
- **成功タスク数** は成功したタスク数です。
- **実行ジョブ数** には、成功したジョブと失敗したジョブが含まれます。

## エージェントの実行ジョブのエキスポート

ユーザーは表示されたサマリー情報を .xlsx 形式の Excel ファイルにエキスポートできます。エキスポート時にはアクティブなフィルターとソート順が適用されます。

## 検索

ユーザーはウィンドウの上部にある検索ボックスから、実行ジョブのサマリーを表示することができます：

- **期間** フィルターボタンは実行ジョブのサマリーの日付範囲を選択します。
- **現在のコントローラー** チェックボックスは、実行ジョブを現在選択されているコントローラーに登録されているエージェントに限定します。

## 詳細検索

詳細検索は、より広い日付範囲、特定のエージェント、コントローラーの検索条件を提供します。

*詳細検索* は、指定された条件でデータをエキスポートすることができます。
