# IDサービス管理 - ブロックリスト


IDサービスは、認証と認可によってJOCコックピットへのアクセスを管理します。[IDサービス管理](/identity-services) を参照してください。

管理者アカウントは、任意の IDサービスのアカウントをブロックリストに追加できます：

- ブロックされたアカウントは JOCコックピットへのアクセスが拒否され、LDAP、OIDC などの ID サービスプロバイダーではブロックされません。
- ブロックされたアカウントは、リストから削除されるまでブロックリストに残ります。

## ブロックリストへのアカウントの追加

*ブロックリスト*画面では、画面右上隅にある *ブロックリストに追加* ボタンからブロックリストにアカウントを追加できます。

ユーザーアカウントは以下の画面からもブロックリストに追加できます：

- [操作ログ - ログイン失敗](/identity-service-faíled-logins) 頻繁にログインに失敗するアカウントが確認された場合、これは攻撃を示している可能性があります。このようなアカウントはブロックリストに追加できます。
- [IDサービス管理 - アクティブセッション](/identity-service-active-sessions) アクティブなセッションのアカウントが不要であることが確認された場合、ブロックリストに追加できます。

どちらの画面も、ブロックリストに単一のアカウントを追加したり、一括操作から選択したアカウントを追加したりできます。

### ブロックリストからのアカウントの削除

*ブロックリスト* 画面では、表示されているアカウントごとに *削除* というアクションメニュー項目があります。

画面右上隅にある　*ブロックリストから削除* を使用すると、選択したアカウントの一括操作が可能です。
