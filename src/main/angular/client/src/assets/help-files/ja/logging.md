# ログ設定

JOCコックピットは、例えば[REST Web Service API](/rest-api) の応答時間を分析する必要が生じた場合、GUI 操作のログを提供します。

ユーザーは必要なログレベルを指定できます：

- **Info**
- **Error**
- **Warn**
- **Debug**

ログを有効にするには、以下のチェックボックスをオンにします：

- **ログ有効化**

ログ出力は *ログ設定* 画面で利用可能になります。ログの設定は永続化され、後のユーザーセッションでも有効になります。

## ログ出力に関する操作

ログ出力が利用可能になると、ビューの右上隅にある関連ボタンから以下の操作を適用できます：

- **クリップボードにコピー** ログ出力をユーザーのクリップボードにコピーします。
- **新規タブで開く** は、関連するログ出力を保持する新しいブラウザタブを開きます。
