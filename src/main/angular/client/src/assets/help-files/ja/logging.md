# ロギング

JOC コックピットは、例えば[REST Web Service API](/rest-api) の応答時間を分析する必要が生じた場合、GUI 操作のロギングを提供します。

ユーザーは必要なログレベルを指定できます：

- **情報
- **エラー
- **Warn**
- **デバッグ

ロギングを有効にするには、以下のチェックボックスをオンにします：

- **ログ有効

ログ出力は*Logging*ビューで利用可能になります。ログの設定は永続化され、後のユーザーセッションでも有効になります。

## ログ出力に関する操作

ログ出力が利用可能になると、ビューの右上隅にある関連ボタンから以下の操作を適用できます：

- **クリップボードにコピー** ログ出力をユーザーのクリップボードにコピーします。
- **新しいタブで開く**は、関連するログ出力を保持する新しいブラウザタブを開きます。
