# 履歴 - オーダー

*履歴 - オーダー* 画面は、オーダーの実行履歴を表示します。これには、関連するオーダーから実行されたワークフローとジョブの実行履歴も含まれます。

*オーダー履歴*は、[クリーンアップサービス](/service-cleanup) によって実行されるデータベース削除の対象となります。

タスク履歴については、[履歴 - タスク](/history-tasks) を参照してください。

## ナビゲーションパネル

左側のパネルでは、ジョブを実行したワークフローやオーダーからタグによるフィルタリングができます。

- **ワークフロータグ** は、[設定 - インベントリー - ワークフロー](/configuration-inventory-workflows) から設定できます。
- **オーダータグ** は、[設定 - インベントリー - スケジュール](/configuration-inventory-schedules) から設定できます。

タグは + と - のアイコンから選択や、検索アイコンで探すことができます。タグの表示は[設定 - JOC](/settings-joc) から有効にする必要があります。

## 履歴パネル

[プロファイル - 設定値](/profile-preferences) から特に指定がない場合、最大5000件まで表示されます。

### オーダー履歴

- **オーダーID** は、オーダーに割り当てられた一意の識別子です。三角アイコンをクリックすると、オーダーの変数とオーダーによって渡されたジョブが表示されます。 
- **ワークフロー** は、オーダーによって実行されたワークフローを示します。
  - ワークフロー名をクリックすると、[ワークフロー](/workflows) 画面に移動します。
  - 鉛筆アイコンをクリックすると、[設定 - インベントリー - ワークフロー](/configuration-inventory-workflows) 画面に移動します。
- **ラベル** は、ワークフローにおけるオーダーの最新位置を示します。ユーザーはワークフローにラベルを割り当てることができます。
- **ステータス** は、オーダーの最新の結果を示します。
  - オーダーが完了した場合、*ステータス* は *成功* または *失敗* となります。
  - オーダーが完了していない場合、*ステータス* は *進行中* となります。
- **オーダーステート**は、オーダーの最新の状態を示します。[オーダー](/order-states) を参照してください。
  - オーダーが完了している場合、*オーダー* は *成功* または *失敗* になります。
  - オーダーが完了していない場合、*オーダー* は *実行中* となります。

### ログ出力

- **オーダーID** をクリックすると、[オーダーログ画面](/order-log) から、ワークフローで実行されたジョブの出力が表示されます。
- **ダウンロード** アイコンをクリックすると、オーダーのログをファイルにダウンロードします。

デフォルトでは、オーダーログの表示は 10MB に制限されています。ユーザーは[設定 - JOC](/settings-joc) 画面から制限を変更できます。

### タスク履歴の操作

オーダー毎にアクションメニューがあり、以下の操作が可能です：

- **ワークフローを無視リストに追加** は、ワークフローのオーダーログを永久に非表示 にします。これは、*オーダー履歴* に入力される周期的なワークフローに便利です。

*無視リスト* はウィンドウ右上のボタンから管理します：

- **無視リスト編集** は、*無視リスト*内のジョブやワークフローを表示し、*無視リスト*から個別に削除することができます。 
- **無視リスト有効化** は、*無視リスト* に追加されたジョブ、または追加されたワークフローに含まれるジョブを非表示にするフィルタリングを有効にします。 *無視リスト* が有効な場合は、関連するボタンが表示されます。
- **無視リスト無効化** を無効にすると、ジョブやワークフローのフィルタリングが無効になります。この操作はアクティブな *無視リスト* に対して有効です。
- **無視リストをリセット** すると、*無視リスト* からジョブやワークフローが削除され、すべてのジョブが表示されます。

## 詳細検索

オーダーの表示を制限するために、ユーザーはウィンドウ上部の検索ボックスを適用することができます。

- **成功**, **失敗**, **進行中** フィルターボタンは、関連する *ステータス* を持つオーダーに表示を制限します。
- **日付範囲** フィルターボタンは、オーダーを表示する日付範囲を選択します。
- **現在のコントローラー** チェックボックスは、現在選択されているコントローラーにオーダーを制限します。
