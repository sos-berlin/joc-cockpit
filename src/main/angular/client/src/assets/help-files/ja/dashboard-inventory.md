# インベントリー

インベントリ*パネルは、使用中のオブジェクトの数に関する情報を提供します。

<img src="dashboard-inventory.png" alt="Inventory" width="330" height="80" />

## インベントリオブジェクト

インベントリサマリーでは、デプロイされたオブジェクトとリリースされたオブジェクトがカウントされます。デプロイもリリースもされていないドラフトオブジェクトはカウントされません。
