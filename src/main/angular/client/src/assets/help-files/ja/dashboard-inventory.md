# ダッシュボード - インベントリー

*インベントリー* ウィジェットは、使用中のオブジェクトの数に関する情報を提供します。

<img src="dashboard-inventory.png" alt="Inventory" width="330" height="80" />

## インベントリーオブジェクト

インベントリーパネルでは、配置されたオブジェクトとリリースされたオブジェクトがカウントされます。配置もリリースもされていないドラフトオブジェクトはカウントされません。
