# タスクログビュー

タスクログビュー*は2-3秒毎に更新される実行ログを提供します。これにより、ジョブの出力をほぼリアルタイムで追うことができます。

## 出力フィルタリング

タスクログビューでは、ウィンドウの上部から様々な条件によるフィルタリングが可能です：

- **Main** はジョブの開始に関する詳細とジョブの開始時のパラメーターを表示します。このような出力は [MAIN] という修飾子で示されます。
- **stdout** はジョブが標準出力チャンネルに書き込んだ出力が [STDOUT] を使って表示されるかどうかを指定します。
-[JS7 - Job API](https://kb.sos-berlin.com/display/JS7/JS7+-+Job+API)**このようなジョブには、*debug* または *trace* 値を持つ *log_level* 引数を追加することができます。デバッグ出力がジョブから利用可能な場合、[DEBUG]修飾子を使用して表示されます。

ジョブが標準エラーチャンネルに書き込んだログ出力はフィルタリングの対象にはならず、[STDERR] 修飾子を使って表示されます。

## ログ出力の表示

ログ出力は到着順に表示されます。

### タイムスタンプ

ログ出力には、さまざまなソースからのタイムスタンプが表示されます：

- **エージェントの時間**：開始]や[終了]などのイベントはエージェントによって作成され、エージェントのリアルタイムクロックを反映します。
- **ジョブの時間**：ジョブの出力は、ジョブが実行されるサーバのタイムゾーン、またはジョブの実装から指定されたタイムゾーンを使用します。

[Profile - Preferences](/profile-preferences) の関連設定が有効な場合、**タスクログビュー* はタイムスタンプをユーザのタイムゾーンに変換します。それ以外はエージェントのタイムゾーンが使用されます。

エージェントのリアルタイムクロックが同期されていない場合、ログ出力のタイムスタンプが不正確になる可能性があります。
