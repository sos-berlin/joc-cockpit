# モニター - コントローラーの可用性

このビューには、コントローラーの可用性のインジケーターが表示されます。

コントローラクラスタを使用している場合は、クラスタの可用性が考慮されます。たとえば、クラスタ内のコントローラインスタンスがメンテナンスのために シャットダウンされ、残りのインスタンスがその負荷を補う場合、可用性は低下しません。

チェックを外すと、接続されているすべてのコントローラの可用性が表示され、チェックを外すと、現在選択されているコントローラのみの情報が表示されます。

ユーザーは、コントローラの可用性の履歴データは、[Cleanup Service](/service-cleanup) によってパージされる可能性があることに注意してください。

## 日付フィルター

パネルの右上隅では、可用性を表示する日付範囲を選択できます：

- **Week**は日付スライダーを1週間単位に切り替えます。
- **月**は1ヶ月単位でスライダーを切り替えます。
- **範囲**は開始日と終了日を指定します。

## 実行時間

指定された期間において、コントローラーが使用可能であると確認されたパーセンテージを表示します。

## 統計

指定された期間における稼働率を棒グラフで表示します。

## 概要

指定された期間における、コントローラと日ごとの稼働率を表示します。

- このグラフでは、Controller の可用性が確認された時間帯を緑色で表示します。 
- 赤色は利用不可を示します。
- 灰色はデータがないことを示します。
