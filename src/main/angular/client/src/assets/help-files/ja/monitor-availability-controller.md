# モニター - コントローラーモニター

この画面には、コントローラーの稼働率のインジケーターが表示されます。

コントローラークラスターを使用している場合は、クラスター全体の稼働率が表示されます。たとえば、クラスター内のコントローラインスタンスがメンテナンスのために シャットダウンされ、残りのインスタンスがその負荷を補う場合、稼働率は低下しません。

*現在のコントローラー* チェックボックスを外すと、接続されているすべてのコントローラー状態が表示され、チェックを外すと、現在選択されているコントローラーのみの情報が表示されます。

[クリーンアップサービス](/service-cleanup) によって、コントローラーモニターの履歴データは、削除されることに注意してください。

## 日付フィルター

パネルの右上隅では、モニターを表示する日付範囲を選択できます：

- **週** は、日付スライダーを1週間単位に切り替えます。
- **月** は、1ヶ月単位でスライダーを切り替えます。
- **カスタム** は、開始日と終了日を指定します。

## 実行時間

指定された期間において、コントローラーが使用可能であると確認されたパーセンテージを表示します。

## 統計情報

指定された期間における稼働率を棒グラフで表示します。

## 概要

指定された期間における、コントローラーと日ごとの稼働率を表示します。

- このグラフでは、コントローラーの稼働が確認された時間帯を緑色で表示します。 
- 赤色は利用不可を示します。
- 灰色はデータがないことを示します。
