# ワークフロー検索

検索ボタンは、ワークフローを検索するために使用します。

- **ユーザー入力**ワークフロー名、タイトル、フォルダー名を入力して検索します。
- ワークフロー状態
  - **同期済** コントローラーに配置されたワークフロー。
  - **一時停止** オーダーを受け付けますが、オーダーの開始はできません。
  - **保留** ワークフローが中断された場合、エージェントからの確認を待ちます。
- ジョブ状態
  - **スキップ** ジョブは実行されず、オーダーは次のジョブに移ります。
  - **停止** ジョブは実行されず、オーダーは停止します。

## メタ文字

- **?**は任意の1文字を置き換えます。
- **\***は0文字以上を置き換えます。

検索は大文字と小文字を区別せず、下記例のように部分的に補足して行われます。

- **rest**は、"pdfNon**Rest**artable" と "**REST**-RunningTaskLog" という名前のワークフローを検索します。
- **re?t**は、"ActivePassiveDi***rect***or "と "JITL-JS7***REST***ClientJob "という名前のワークフローを検索します。
- **re\*t**は、"pdSQLExecuto**rExt**ractJSON "と "pdu**Reset**Subagent "という名前のワークフローを見つけます。

## 詳細

この機能は次のリンクをクリックすると利用できます:<br/>**> 詳細**

### 属性検索

詳細では、オブジェクトの各属性で検索することができます：

- **エージェント名**は、指定されたエージェントで実行されたジョブを含むワークフローを返します。
- **ジョブ数** は、**From** 条件で指定された最小数のジョブを使用するワークフローを返します。**To**条件と一緒に使用した場合、*From*から*To*までの範囲のジョブ数を含むワークフローが返されます。**To**条件のみを使用した場合、*To*条件を超えるジョブ数を含まないワークフローが返されます。
- **ジョブ名**は、指定された名前に一致するジョブを含むワークフローを返します。**ジョブ名**の*完全一致*チェックボックスを使用する場合、入力された検索語は大文字と小文字の区別を含むジョブ名と完全に一致する必要があります。

### 依存関係検索

検索メタ文字 **\*** は、例えばロックがどのような名前であっても、依存関係が検索されることを指定するために使用されます：

- ロック項目の**\***メタ文字は、ロックを参照しているワークフローを返します、
- ファイルオーダーソース項目**\***のメタ文字は、ファイルオーダーソースによって参照されるワークフローを返します。
