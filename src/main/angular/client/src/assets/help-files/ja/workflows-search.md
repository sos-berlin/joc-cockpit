# ワークフロー検索

ワークフロー検索は、ワークフローを検索するために使用されます。

- **ワークフロー検索は、ワークフローを検索する際に使用します、
- ワークフローの可用性
  - **ワークフローはコントローラにデプロイされます。
  - **一時停止中（Suspended）*** ワークフローは凍結され、オーダーを受け付けますが、オーダーの開始はできません。
  - **ワークフローが中断された場合、エージェントからの確認を待ちます。
- ジョブの可用性
  - **スキップされた**ジョブは、オーダーが通過しても実行されません。
  - **停止中(Stopped)*** ジョブは到着したオーダーを一時停止します。

## メタ文字

- **メタ文字は任意の1文字を置き換えます。
- **\*** メタ文字は0文字以上を置き換えます。

検索は大文字と小文字を区別せず、部分的に修飾して行われます。

- **rest**は、"pdfNon**Rest**artable" と "**REST**-RunningTaskLog" という名前のワークフローを検索します。
- **re?t**は、"ActivePassiveDi***rect***or "と "JITL-JS7***REST***ClientJob "という名前のワークフローを見つけます。
- re**t**は、"pdSQLExecuto**rExt**ractJSON "と "pdu**Reset**Subagent "という名前のワークフローを見つけます。

## 詳細検索

この機能は次のリンクをクリックすると利用できます:<br/>**&gt; Advanced**

### 属性検索

詳細検索では、オブジェクトの属性で検索することができます：

- **エージェント名**は、指定されたエージェントで実行されたジョブを含むワークフローを返します。
- **Count Jobs** は、**From** 条件で指定された最小数のジョブを使用するワークフローを返します。To**項と一緒に使用した場合、*From*から*To*までの範囲のジョブ数を含むワークフローが返されます。To**項のみを使用した場合、*To*項を超えるジョブ数を含まないワークフローが返されます。
- **ジョブ名**は、指定された名前に一致するジョブを含むワークフローを返します。ジョブ名**の*完全一致*チェックボックスを使用する場合、入力された検索語は大文字と小文字の区別を含むジョブ名と完全に一致する必要があります。

### 依存関係検索

検索メタ文字 ******* は、例えばリソースロックがどのような名前であっても、依存関係が検索されることを指定するために使用されます：

- **\*** Resource Locks*** メタ文字は、リソースロックを使用しているワークフローを返します、
- **\*** のメタ文字は、ファイルオーダーソースによって参照されるワークフローを返します。
