# プロファイル - お気に入り管理

*プロファイル - お気に入り管理* タブでは、別々のタブからお気に入りのエントリーを指定することができます。

- *エージェント* タブは、ジョブに割り当てられた *エージェント* を選択します。
- *ファセット* タブは、オーダー変数へのユーザー入力をルール化します。

詳細は[JS7 - Inventory Favorites](https://kb.sos-berlin.com/display/JS7/JS7+-+Inventory+Favorites) および[JS7 - Profiles - Favorite Management](https://kb.sos-berlin.com/display/JS7/JS7+-+Profiles+-+Favorite+Management) を参照してください。

## お気に入りエージェント

ユーザはー、[スタンドアロンエージェント](/initial-operation-register-agent-standalone) と[サブエージェントクラスター](/initial-operation-register-agent-subagent-cluster) から、自分の環境に、多くのエージェントの長いリストをスクロールする代わりに、割り当て可能なエージェントのリストの一番上に来るように、*お気に入りのエージェント*を指定することができます。

### お気に入りエージェント追加

*お気に入りエージェント* は、[ジョブ定義 - インベントリー - ワークフロー - ジョブプロパティ](/configuration-inventory-workflow-job-properties) タブでエージェントを割り当てる際に直接追加することができます。エージェント名の右側にある星形アイコンをクリックすると、選択したエージェントがお気に入りリストに追加されます。

ユーザーは、星形アイコンを使って、*プロファイル* から *お気に入りエージェント* を追加することができます。これにより、追加するエージェントを選択することができます。

### お気に入りエージェントのリスト

お気に入りエージェントのリストは、*お気に入り共有* ボタンから制限することができます。これにより、他のユーザーによって共有されたお気に入りにリストが制限されます。

*検索* 入力フィールドにエージェント名から検索される文字列を指定することで、さらにリストを制限することができます。検索は左右の文字切り捨てでも可能です。

ユーザーは、*お気に入りエージェント* を前または後に移動することで、リスト順を指定することができます。

### お気に入りエージェントの操作

*お気に入りエージェント* のアクションメニューから以下の操作が可能です：

- **編集** は、エージェントを切り替えることができます。
- **共有** は、お気に入りエージェントを他のユーザーと共有することができます。共有されたお気に入りには関連アイコンが表示されます。
- **削除** は、お気に入りリストから削除します。

## お気に入りファセット
ユーザーは、[ジョブ定義 - インベントリー - ワークフロー](/configuration-inventory-workflows)から文字列データ型のオーダー変数を指定する場合、*ファセット* を適用することによってユーザー入力を制限することができます。 ユーザー入力が正規表現に一致するかを *ファセット* によって確認できます。

*ファセット*をオーダー変数に割り当てる場合、長いリストをスクロールする代わりに、ユーザーはリストの最上位に表示されるお気に入りの* ファセット*を指定できます。

### お気に入りファセットの追加

[ジョブ定義 - インベントリー - ワークフロー](/configuration-inventory-workflows) 画面でオーダー変数を指定する際、*お気に入りファセット* を直接追加することができます。オーダー変数の *ファセット入力モード* リンクをクリックすると、関連する *ファセット* がお気に入りリストに追加されます。

ユーザーは、*プロファイル* の *お気に入り追加* ボタンを使用して、*お気に入りファセット* を追加できます。これにより、追加する*ファセット* 名と関連する正規表現を指定することができます。

### お気に入りファセットのリスト

*お気に入り共有* ボタンから、*お気に入りファセット* のリストを制限することができます。これにより、リストが他のユーザーによって共有されたお気に入りに制限されます。

*検索* 入力フィールドにファセット名から検索される文字列を指定することで、さらにリストを制限することができます。検索は左右の文字切り捨てでも可能です。

ユーザーは、*お気に入りファセット* を前または後に移動することで、リスト順を指定できます。

### お気に入りファセットに対する操作

お気に入りファセットのアクションメニューから以下の操作が可能です：

- **編集** は、ファセットの名前と正規表現を変更することができます。
- **共有** は、お気に入りを他のユーザーと共有することができます。共有されたお気に入りには、関連アイコンが表示されます。
- **削除** は、お気に入りリストから関連するファセットを削除します。
