# ダッシュボード - ワークフロー

*ワークフロー* ウィジェットはワークフローのステータスに関する情報を提供します。

<img src="dashboard-workflows.png" alt="Workflows" width="330" height="80" />

## ワークフローステータス

ワークフローは以下のステータスから表示されます：

- **同期済** ワークフローは配置され、コントローラーおよびエージェント利用可能です。
- **未同期** ワークフローはコントローラーとエージェントに配置されておらず、インベントリーからのみ利用可能です。
- **一時停止中** ワークフローは一時停止され、オーダーを受け付けますが、ワークフローが再開されるまでオーダーの開始を許可しません。
- **保留中** ワークフローは、ワークフローが中断されていることを 1 つ以上のエージェントが確認するまで待機します。

表示されたワークフロー数をクリックすると、*ワークフロー* 画面に移動し、関連するワークフローが表示されます。
