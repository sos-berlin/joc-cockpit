# ♪ワークフロー

ワークフロー*パネルはワークフローのステータスに関する情報を提供します。

<img src="dashboard-workflows.png" alt="Workflows" width="330" height="80" />

## ワークフローステータス

ワークフローは以下のステータスから表示されます：

- **Synchronized** ワークフローはデプロイされ、Controller および Agent で利用可能です。
- **ワークフローはコントローラとエージェントにデプロイされておらず、インベントリからのみ利用可能です。
- **一時停止中** ワークフローは凍結され、オーダーを受け付けますが、ワークフローが再開されるまでオーダーの開始を許可しません。
- **ワークフローは、ワークフローが中断されていることを 1 つ以上のエージェントが確認するまで待機します。

表示されたワークフロー数をクリックすると、*ワークフロー*ビューに移動し、関連するフィルタを使用したワークフローが表示されます。
