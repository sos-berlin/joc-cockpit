# ジョブ定義 - インベントリー - 操作 - タグ管理

JS7はワークフロー、ジョブ、オーダーに[JS7 - Tagging](https://kb.sos-berlin.com/display/JS7/JS7+-+Tagging)機能を提供します 。

- タグは JOCコックピットの *ジョブ定義* 画面で利用可能なインベントリーから管理されます。
  - ワークフローには任意の数のタグが割り当てられます。
  - スケジュールには、[デイリープランサービス](/service-daily-plan) によって作成されたオーダーに追加されるタグがいくつでも割り当てられます。
- タグは、アドホックオーダーを手動で追加するときにも適用できます。
- タグは *ワークフロー*、*オーダーサマリー*、*実行計画*、*履歴* 画面でワークフローとオーダーのフィルタリングに使用できます。

## ワークフローのタグ管理

ワークフローのタグを追加、更新、削除することができます。

<img src="tags-manage-object.png" alt="Manage Tags for Workflow" width="600" height="125" />

タグを入力する際、既存のタグのリストが表示されます。それ以外に、ユーザーは新しいタグの名前を入力することができます。タグ名の大文字と小文字は区別されます。

## フォルダーからワークフローのタグを管理

ユーザーは、選択したフォルダーおよびサブフォルダーで利用可能なすべてのワークフローのタグを、再帰的に追加、更新、削除できます。

<img src="tags-manage-folder.png" alt="Manage Tags for Workflows from Folders" width="600" height="210" />

タグの *リスト* には、タグ名とタグが割り当てられているワークフローが表示されます。ユーザーはタグの割り当てを削除できます。
