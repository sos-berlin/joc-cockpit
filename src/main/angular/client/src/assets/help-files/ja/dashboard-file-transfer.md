# ファイル転送

ファイル転送*パネルには、YADE 管理ファイル転送を使用した過去の転送の概要が表示されます。

<img src="dashboard-file-transfer.png" alt="File Transfer" width="320" height="90" />

## ファイル転送履歴

履歴の要約は、成功した転送と失敗した転送をカウントします。

- **成功した**転送には、成功したジョブと、再試行などによって回復した失敗したジョブが含まれます。
- **失敗した**転送は、永久に失敗したジョブを示します。

表示された番号をクリックすると、転送の詳細を表示する*ファイル転送履歴*ビューに移動します。

## フィルター

パネルの右上隅にあるドロップダウンボタンは、日付範囲から過去の転送ジョブを選択することができます：

- **本日**のジョブはユーザーのプロフィールのタイムゾーンから計算される現在の日に関連します。
- **直近1時間**は直近1時間に完了したジョブを含みます。
- **直近12時間以内** には、直近12時間以内に完了したジョブが含まれます。
- **直近24時間以内** には、直近24時間以内に完了したジョブが含まれます。
- **直近7日間**には、直近7日間に完了したお仕事が含まれます。
