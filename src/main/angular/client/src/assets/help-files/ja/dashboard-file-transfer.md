# ダッシュボード - ファイル転送サマリー

*ファイル転送サマリー* ウィジェットには、YADE ファイル転送を使用したジョブの過去の転送の概要が表示されます。

<img src="dashboard-file-transfer.png" alt="File Transfer" width="320" height="90" />

## ファイル転送履歴

履歴は、成功した転送と失敗した転送をカウントします。

- **成功** は、成功したジョブと、再試行などによって回復した失敗したジョブが含まれます。
- **失敗** は、永久に失敗したジョブを示します。

表示された番号をクリックすると、転送の詳細を表示する *ファイル転送履歴* 画面に移動します。


## 検索ボタン

パネルの右上にあるドロップダウンボタンは、日付範囲から過去の転送ジョブを選択することができます：

- **本日** は、ユーザープロファイルのタイムゾーンから計算される現在の日の履歴を表示します。
- **最新1時間** は、直近1時間に完了したジョブです。
- **最新12時間** は、直近12時間以内に完了したジョブが含まれます。
- **最新24時間** は、直近24時間以内に完了したジョブが含まれます。
- **最新7日間** は、直近7日間に完了したジョブが含まれます。
