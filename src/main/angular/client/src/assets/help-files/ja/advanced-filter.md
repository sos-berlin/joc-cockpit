# 詳細検索

*詳細検索* は、検索結果をいくつかの画面で制限することができます。

## カスタマイズ

- ユーザーは検索条件を保存するカスタマイズ名を指定し、後で再利用することができます。
- カスタマイズはほとんどの画面で利用可能で、メインメニューの下にあるドロップダウンボタンから有効化できます。

## 検索条件

検索条件は加算式で、指定されたすべての条件に一致する項目が検索結果に追加されます。

入力フィールドからの検索は, 大文字小文字を区別せずに行われます.

- **rest** は、"pdfNon***Rest***artable" と "**REST**-RunningTaskLog" という名前のオブジェクトを見つけます。
- **re?t** は、"ActivePassiveDi**rect**or" と "JITL-JS7**REST**ClientJob" という名前のオブジェクトを見つけます。
- re***t** は、"pdSQLExecuto**rExt**ractJSON "と "pdu**Reset**Subagent "という名前のオブジェクトを見つけます。

### 検索結果

検索結果は、利用可能な一括操作に対して表示されます。
