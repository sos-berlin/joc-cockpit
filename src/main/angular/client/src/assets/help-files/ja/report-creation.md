# レポート - 実行レポート

[JS7 - Reports](https://kb.sos-berlin.com/display/JS7/JS7+-+Reports) は過去のワークフローとジョブの実行状況から得られるレポートを作成します：

- レポートは[ジョブ定義 - インベントリー - レポート](/configuration-inventory-reports) で定義済みの[レポートテンプレート](/report-templates) から作成されます。
- ユーザーは *実行レポート* 画面または[JS7 - Reports - Automation](https://kb.sos-berlin.com/display/JS7/JS7+-+Reports+-+Automation) から実行できます。[レポート - レポート実行履歴](/report-run-history) 画面は、過去のレポート実行に関する情報を確認できます。
- レポートは[レポート](/reports) 画面で表示されます。

## レポート設定の表示

レポートは[ジョブ定義 - インベントリー - レポート](/configuration-inventory-reports) 画面で設定できます。

レポートの設定は、

- インベトリーの *実行計画 - レポートフォルダー* から作成できます。
- レポートを選択すると詳細を表示する *レポート設定* が表示されます。

### レポートフォルダー

この画面では、レポート設定をもつフォルダーを表示できます：

- フォルダーを選択すると、利用可能なレポートが表示されます。
- フォルダーにマウスカーソルを合わせると表示される三角アイコンをクリックすると、そのフォルダーおよびすべてのサブフォルダーのレポート設定が表示されます。

### レポート設定画面

この画面には、現在選択されているフォルダから利用可能なレポート設定が表示されます。レポート設定は以下の項目で表示されます：

- **名称**は[ジョブ定義 - インベントリー - レポート](/configuration-inventory-reports) 画面で指定したレポート設定の一意の名前を表示します。
- **テンプレート**は使用している[レポートテンプレート](/report-templates) を表示します。
- **周期**は、*レポート期間* を分割した *レポート周期* を表示します。
- **開始月**、**終了月**は、*レポート期間* の開始月と終了月を示します。

## レポートの作成

レポートはレポート設定のアクションメニューから *レポート実行* 操作を使って作成できます。

ユーザーは複数のレポート設定を選択し、画面右上のボタンから *レポート実行* の一括操作が可能です。
