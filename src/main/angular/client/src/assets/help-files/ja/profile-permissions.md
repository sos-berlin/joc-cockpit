# プロファイル - パーミッション

*プロファイル - パーミッション* 画面では、ユーザーアカウントに割り当てられたロールと権限が表示されます。

## ロール

ユーザーアカウントに割り当てられたロールが表示されます。ロールの割り当てから権限が適用されます。

## 権限

### JOCコックピット

すべての JOCコックピットに対する権限が表示されます。
 
### コントローラー

コントローラーインスタンスごとに権限が表示されます。関連するコントローラーに特定の権限が設定されていない場合は、コントローラーの *default* 権限が適用されます。
