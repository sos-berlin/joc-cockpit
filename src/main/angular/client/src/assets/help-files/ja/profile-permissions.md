# プロフィール - 権限

プロファイル-パーミッション*ページでは、ユーザアカウントに割り当てられたロールの結果としてのパーミッションが報告されます。

## ロール

ユーザアカウントに割り当てられたロールが表示されます。結果の権限はロールの割り当てからマージされます。

## 権限

### JOCコックピット権限

すべての JOC コックピット・インスタンスに対するパーミッションが表示されます。
 
### コントローラ権限

許可はコントローラインスタンスごとに表示されます。関連するControllerに特定のパーミッションが設定されていない場合は、Controllerの*デフォルト*パーミッションが適用されます。
