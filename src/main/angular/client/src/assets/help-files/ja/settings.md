# 設定

JOC Cockpitはランタイム[JS7 - Settings](https://kb.sos-berlin.com/display/JS7/JS7+-+Settings) 、GUIを使用して変更でき、JOC Cockpitを再起動することなく即座に適用されます。

メニューバーの ![wheel icon](assets/images/wheel.png) アイコンから *設定* ページにアクセスできます。
