# 設定

JOCコックピットは[JS7 - 設定](https://kb.sos-berlin.com/display/JS7/JS7+-+Settings) 画面を使用して変更でき、JOC Cockpitを再起動することなく即座に適用されます。

メニューバーの右側のホイールアイコンから *設定* ページにアクセスできます。
