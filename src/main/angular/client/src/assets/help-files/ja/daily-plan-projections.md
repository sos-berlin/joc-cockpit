# 実行計画 - プロジェクション

実行計画では、数日前にコントローラーとエージェントに提出されるオー ダーを保持します。また、6ヶ月先までのオーダー開始時間のプロジェクションも可能です。

予測期間を長くしたい場合は、[設定 - Daily Plan](/settings-daily-plan) 画面で設定を変更できます。

### 日付、期間、タイムゾーン

プロジェクションは、カレンダーの日付ではなく、実行計画の日付に関連しています。 

- 期間
  - 実行計画の24時間期間が午前0時に開始する場合は、暦日と一致します。
  - 実行計画が日中に開始する場合、24時間の期間は2つの暦日と重なります。
- タイムゾーン
  - スケジュールが実行計画と異なるタイムゾーンを使用している場合、オーダー開始時間が前日または翌日と重なることがあります。このようなオーダーは、実行計画の日付で表示されますが、開始時刻は異なる日付で表示されます。
  - タイムゾーンを使用すると、24時間の実行計画に加え、-14時間や+12時間の開始時間が発生することがあります。驚くことに、1日は24時間ではなく、最大50時間です。地球の自転に依存するため、1日の長さは常に24時間です。しかし、与えられたタイムゾーンでは、地球上のすべての可能な時間を含むように50時間の範囲があります。

すべての日付と時刻は、ユーザーのプロファイルで指定されたタイムゾーンで表示されます。

### 表示オプション

ユーザーは、ウィンドウの左上隅にあるリンクを使用して、プロジェクションされたオーダーの *毎月* 表示と *毎年* 表示を切り替えることができます。

各日ごとに、月中の日数とプロジェクションオーダー数が表示されます：

- **緑色のオーダー**： コントローラーとエージェントに登録済みのオーダーです。
- **オレンジ色のオーダー**：開始時間ルールに基づいて計算された、予測オーダーを表します。
- **プロジェクション反転**：
  - **チェックなし**：オーダーが計算された日数とオーダー数が表示されます。ユーザーは、個々の日をクリックして、オーダーの開始時間を確認できます。
  - **チェックあり**：プロジェクションを反転させると、オーダーを作成しないスケジュールが存在する日が強調表示されます。関連する日をクリックすると、オーダーのないスケジュールが表示されます。

### 詳細検索

ワークフローやスケジュールを保存している特定のフォルダに限定してオーダーを表示することができます。

## プロジェクションの操作

### プロジェクションの作成

- プロジェクションは、デイリープランサービスによって日次実行中に計算されます。日中の実行計画の変更は反映されません。
- ユーザーは、*再作成* ボタンからオンデマンドでプロジェクションを再作成できます。
- *確認時刻*は、現在のプロジェクションの作成日を示します。

### プロジェクションのエキスポート

プロジェクションは、X 軸に実行計画日、Y 軸にワークフローとスケジュールを持つ .xlsx ファイルにエクスポートできます。

- *エキスポート* のショートカットは、ウィンドウに表示されているオーダーをエクスポートします。 
- *エキスポート* ボタンをクリックすると、エキスポートするオーダーを選択できます：
  - **開始日**、**終了日**：オーダーをエキスポートする実行計画の最初と最後の日付です。
  - **ワークフロー**、**スケジュール**： 特定のスケジュールとワークフローにエキスポートを制限できます。
  - **プロジェクション反転**： 
    - **チェックなし**：オーダーが計算された日付をエキスポートします。
    - **チェックあり**：オーダーが計算されない日付をエキスポートします。これは、非営業日が考慮されるかどうかを確認するために使用できます。
