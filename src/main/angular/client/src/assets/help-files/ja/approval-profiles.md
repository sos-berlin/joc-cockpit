# 承認者プロファイル

[JS7 - Approval Process](https://kb.sos-berlin.com/display/JS7/JS7+-+Approval+Process) は、オーダーの追加やキャンセルなど、他のユーザーの承認が必要な操作のために提供されます。これには、スケジュールオブジェクトを変更する操作も含まれます。

承認プロセスには、以下の役割が含まれます：

- *承認依頼者*は、承認が必要な操作の承認を依頼します。
- *承認者*は、承認依頼を承認または拒否します。

承認プロセスの基本機能は次のとおりです：

- 4-eyesの原則（２名認証）のために、*承認者* は、*承認依頼者* が操作を実行する前に、*承認依頼者* のアカウント、ロール、および権限の範囲を確認する必要があります。 
- 保留中の承認依頼を追跡します。
- 複数の *承認者* の代替候補を複数提供します。

## 承認者プロファイルの一覧

承認者プロファイルを追加するには、ビューの右上にある *新規承認者* ボタンをクリックします。

承認者プロファイルの一覧には、次の項目が表示されます：

- 承認者の  ***アカウント名**、**姓**、**名** を指定します。
- **メール** は、 *承認者* のメールアドレスを指定します。指定した場合、*承認者* は[承認依頼](/approval-requests) の受信通知をメールで受け取ります。

## 承認者プロファイルの操作

承認者プロファイルのアクションメニューで、次の操作を行います：

- **承認プロファイル** の項目を更新します。
