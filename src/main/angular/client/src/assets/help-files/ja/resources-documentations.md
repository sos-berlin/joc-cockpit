# ドキュメント

Documentations*ビューは、Documentationsの使用に関する情報を表示し、Documentationsの管理を可能にします。

Documentationsはワークフローやジョブから参照され、[Workflows](/workflows) ビューで表示されます。

## ナビゲーションパネル

左側のパネルにはドキュメントを保持するインベントリフォルダのツリーが表示されます。

- フォルダをクリックすると、そのフォルダのドキュメンテーションが表示されます。
- フォルダをクリックすると、そのフォルダのドキュメンテーションが表示されます。

クイック検索] アイコンは、ユーザー入力に基づいてドキュメンテーションを検索します：

- Test**と入力すると、*test-documentation-1*や*test-documentation-2*といった名前のドキュメントが表示されます。 
- Test**と入力すると、*test-documentation-1* や *my-TEST-documentation-2* といった名前のドキュメントが表示されます。

## ドキュメンテーションパネル

### ドキュメントの表示

以下の情報が表示されます：

- **Name**はドキュメンテーションの固有名です。
- **参照**は、アップロードされたドキュメント・ファイルの名前です。
- **Type**は、サポートされているタイプ*Text*、*Markdwon*、*HTML*、*XML*、*PDF*などのいずれかです。
- **Modified**最終更新日です。

### ドキュメントに対する操作

ドキュメントのアクションメニューでは、以下の操作が可能です：

- **ドキュメントを編集する**参照されているファイルを変更します。
- **ドキュメンテーションのプレビュー**は、ドキュメンテーションを表示するために、関連するMIMEタイプが割り当てられたアプリケーションを開きます。
- **Export(エクスポート)**は、ドキュメントを.zipアーカイブファイルにエクスポートします。
- **ドキュメントを削除します。
- **使用状況を表示する**」は、ドキュメントを参照しているワークフローを表示します。

ウィンドウの上部には、*Add Documentation* ボタンがあります：

- **ドキュメントのアップロード**では、アップロードするファイルを選択できます。
- **インベントリ内のドキュメントのフォルダと名前を指定します。
