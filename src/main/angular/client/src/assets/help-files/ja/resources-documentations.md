# リソース - ドキュメント

*ドキュメント* 画面は、ドキュメントの使用状況を表示し、管理を可能にします。

ドキュメントはワークフローやジョブから参照され、[ワークフロー](/workflows) 画面から表示されます。

## ナビゲーションパネル

左側のパネルにはドキュメントを保持するインベントリーフォルダのツリーが表示されます。

- フォルダーをクリックすると、そのフォルダーのドキュメントが表示されます。
- フォルダーにマウスカーソルを合わせると表示される三角アイコンをクリックすると、そのフォルダーおよびすべてのサブフォルダーのドキュメントが表示されます。

検索アイコンは、ユーザー入力に基づいてドキュメントを検索します：

- **Test** は、*test-documentation-1*や*test-documentation-2*といった名前のドキュメントを検索します。 
- **\*Test** は、*test-documentation-1* や *my-TEST-documentation-2* といった名前のドキュメントを検索します。

## ドキュメント画面

### ドキュメントの表示

以下の情報が表示されます：

- **ドキュメント名** はドキュメントの一意な名称です。
- **参照** は、アップロードされたドキュメントファイルの名称です。
- **タイプ** は、サポートされているタイプ *Text*、*Markdwon*、*HTML*、*XML*、*PDF*などのいずれかです。
- **最新更新日時** は、最終更新日です。

### ドキュメントの操作

ドキュメントのアクションメニューでは、以下の操作が可能です：

- **ドキュメント編集** は、参照されているファイルを編集します。
- **ドキュメントプレビュー** は、ドキュメントを表示するために、関連するMIMEタイプが割り当てられたアプリケーションを開きます。
- **エキスポート** は、ドキュメントを.zipファイルにエキスポートします。
- **削除** は、ドキュメントを削除します。
- **使用表示**」は、ドキュメントを参照しているワークフローを表示します。

ウィンドウの上部には、*ドキュメントインポート* ボタンがあります：

- **アップロードファイルを選択**では、アップロードするファイルを選択できます。
- **パス** インベントリー内のドキュメントのフォルダと名前を指定します。
