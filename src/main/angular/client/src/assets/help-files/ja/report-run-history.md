# レポートの実行履歴

[JS7 - Reports](https://kb.sos-berlin.com/display/JS7/JS7+-+Reports) はワークフローとジョブの過去の実行履歴を提供します：

- レポートは、[Configuration - Inventory - Reports](/configuration-inventory-reports) からパラメータ化された事前定義[Report Templates](/report-templates) から作成されます。
- ユーザーは[Report Creation](/report-creation) ビューまたは[JS7 - Reports - Automation](https://kb.sos-berlin.com/display/JS7/JS7+-+Reports+-+Automation) からレポートを実行します。レポート実行履歴* ビューは、過去のレポート実行に関する証拠を提供します。
- レポートは[Reports](/reports) ビューで可視化されます。

## レポート実行履歴の表示

レポート実行履歴は以下のプロパティから表示されます：

- **Name**は、[Configuration - Inventory - Reports](/configuration-inventory-reports) ビューで指定されたレポート構成の一意の 名前を示します。
- **コントローラ ID** は、レポートが作成されたコントローラを示します。
- **レポートテンプレート**は、使用中の[Report Template](/report-templates) を示します。
- **Status**は*running*、*successful*、*failed*のいずれかを示します。
- 頻度**は、*レポート期間*を等しい単位に分割する*レポート頻度**を示します。
- **レポート数** は、レポート構成で作成されたレポートの数を示します。
- **エラーメッセージ**は、レポート実行の*ステータス*が*失敗*した場合に発生したエラーを示します。
- **開始月**、終了月**は*レポート期間*の開始と終了を示します。
