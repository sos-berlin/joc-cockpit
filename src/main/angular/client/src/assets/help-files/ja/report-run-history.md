# レポート- レポート実行履歴

[JS7 - Reports](https://kb.sos-berlin.com/display/JS7/JS7+-+Reports) はレポートの過去の実行履歴を表示します：

- レポートは、[ジョブ定義 - インベントリー - レポート](/configuration-inventory-reports) で事前定義した[レポートテンプレート](/report-templates) から作成されます。
- ユーザーは[実行レポート](/report-creation) 画面または[JS7 - Reports - Automation](https://kb.sos-berlin.com/display/JS7/JS7+-+Reports+-+Automation) からレポートを実行します。*レポート実行履歴* 画面は、過去のレポート実行に関する情報を提供します。
- レポートは[レポート](/reports) 画面に表示れます。

## レポート実行履歴の表示

レポート実行履歴は以下の項目から表示されます：

- **名称**は、[ジョブ定義 - インベントリー - レポート](/configuration-inventory-reports) 画面で指定されたレポートの一意の名前を表示します。
- **コントローラーID** は、レポートが作成されたコントローラを表示します。
- **テンプレート**は、使用している[レポートテンプレート](/report-templates) を表示します。
- **ステータス**は *実行中*、*完了*、*失敗* のいずれかを表示します。
- **周期**は、*レポート期間* を分割した *レポート周期* を表示します。
- **レポートNo.** は、作成されたレポートの数を表示します。
- **エラーテキスト**は、レポート実行の *ステータス* が *失敗* した場合に発生したエラーメッセージを表示します。
- **開始月**、**終了月** は *レポート期間* の開始月と終了月を表示します。
