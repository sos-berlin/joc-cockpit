# カレンダー検索

*カレンダー検索*は、次のような条件に基づいてカレンダーを検索するために使用します。

- **ユーザー入力** 与えられた名前またはタイトルにマッチするもの。

## メタ文字

- **メタ文字は任意の1文字を置き換えます。
- **\*** メタ文字は0文字以上を置き換えます。

検索は大文字小文字を区別せず、部分的に修飾して行われます。

- **test**は "My-**Test**-Board-1 "と "**TEST**-Board-2 "という名前のカレンダーを検索します。
- **te?t**は "Global-**Test**-Board-1 "と "**TEXT**-Board-2 "という名前のカレンダーを検索します。
- te**t**は "My-**tExt***-Board-1 "と "My-**Terminat***ing-Board-2 "という名前のカレンダーを見つけます。

## 高度な検索

この機能は次のリンクをクリックすると利用できます:<br/>**&gt; Advanced**

### 属性検索

詳細検索では、オブジェクトの属性で検索することができます：

- **エージェント名**は、指定されたエージェントで実行されたジョブを含むワークフローのカレンダーを返します。
- **Count Jobs** は、**From** 条件で指定された最小数のジョブを使用するワークフローのカレンダーを返します。To**項と一緒に使用した場合、*From*から*To*までの範囲のジョブ数を含むワークフローが返されます。To**項のみを使用した場合、*To*項を超えるジョブ数を含まないワークフローが返されます。
- **ジョブ名**は、指定された名前に一致するジョブを含むワークフローのカレンダーを返します。ジョブ名**の*完全一致*チェックボックスを使用する場合、入力された検索語は大文字と小文字を区別するスペルも含め、ジョブ名と完全に一致する必要があります。

### 依存関係検索

検索メタ文字 ******* は、例えばリソースロックがどのような名前であっても、依存関係が検索されることを指定するために使用されます：

- **\*** のメタ文字は、リソースロックを使用するワークフローのカレンダーを返します、
- **\*** ファイルオーダーソース**のメタ文字は、ファイルオーダーソースによって参照されるワークフローのカレンダーを返します。
